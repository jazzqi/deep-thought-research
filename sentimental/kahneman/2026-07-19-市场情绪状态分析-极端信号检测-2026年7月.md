# 市场情绪状态分析：极端信号检测报告

**分析日期**：2026年7月19日
**分析师**：Kahneman（专业分析师）
**置信度**：85%

---

## 摘要

基于2026年6-7月最新可得数据，中国市场情绪处于**极度悲观**状态。三重核心经济指标——固定资产投资、新增贷款、零售销售——同步录得极端异常值，叠加全球科技板块资金大幅外逃，构成了多重极端信号共振的局面。债券市场定价已提前反映衰退预期。

---

## 一、极端信号详细检测

### 🔴 信号1：固定资产投资同比 -15.6%（2026年6月）

| 维度 | 评估 |
|------|------|
| **数值** | -15.6% |
| **数据时间** | 2026年6月 |
| **历史对比** | 中国FAI历年极少出现超过-10%的跌幅，-15.6%属于极端异常 |
| **含义** | 企业投资意愿崩塌，私营部门和地方政府的投资急剧收缩 |
| **严重程度** | ★★★★★（极严重） |

固定资产投资（FAI）录得-15.6%的同比跌幅，这是在经济数据中极其罕见的数字。即使在全球金融危机期间，中国FAI也曾依靠刺激政策维持正增长。这一数字暗示：

- 地方政府财政压力导致基建投资大幅缩减
- 房地产行业投资持续萎缩（新开工面积数据缺失但可推断为大幅负增长）
- 制造业企业因需求不足和利润恶化而推迟或取消资本开支计划

### 🔴 信号2：新增贷款同比 -25.2%（2026年6月）

| 维度 | 评估 |
|------|------|
| **数值** | -25.21% |
| **数据时间** | 2026年6月 |
| **历史对比** | 信贷脉冲通常为正，-25%属于极端萎缩 |
| **含义** | 信贷需求断崖式下降，货币传导机制失效 |
| **严重程度** | ★★★★★（极严重） |

新增贷款同比下滑25.2%，这是一个信贷市场"冻结"的信号。与FAI的-15.6%结合看，反映出"投资意愿低→信贷需求不足→银行惜贷→企业更难融资→进一步压缩投资"的负反馈循环。

值得注意的是，社融存量同比（aggregate_financing_yoy）为6245（单位不清，可能为亿元），但增量急剧下降意味着存量也在边际恶化。

### 🟠 信号3：零售销售同比仅1.0%（2026年6月）

| 维度 | 评估 |
|------|------|
| **数值** | 1.0% |
| **数据时间** | 2026年6月 |
| **历史对比** | 2023年疫后复苏期间曾达5-10%，1%属于极低水平 |
| **含义** | 消费接近零增长，内需极度疲弱 |
| **严重程度** | ★★★★☆（严重） |

零售销售同比仅1.0%，考虑到名义GDP增速在4.7%左右，实际消费增长可能为负。这与消费者信心指数89.9（低于100基准线）完全吻合——消费者对未来收入预期悲观，倾向于增加储蓄、减少消费。

### 🟠 信号4：消费者信心指数 89.9（2026年5月）

低于100的基准线，表明整体消费者情绪处于悲观区间。这是上述三个同步指标的心理根源——信心不足导致不消费、不投资、不借贷。

### 🟡 信号5：全球科技板块资金外逃

新闻数据显示："科技股遭持续抛售，科技板块ETF过去一个月净流出87亿美元"。"半导体板块本周重挫"，瑞银与巴克莱仍维持看多判断，但这可能只是卖方分析师在下跌中的被动维护。

## 二、交叉验证：债券市场定价

| 指标 | 数值 | 含义 |
|------|------|------|
| 10年期国债收益率 | 1.7404% | 处于历史低位，反映经济增长悲观预期 |
| 2年期国债收益率 | 1.2645% | 短端利率走低表明市场预期货币政策将持续宽松 |
| 期限利差（10Y-2Y） | 0.4759% | 利差为正但不算极端，表明并未出现倒挂危机，但长端收益率极低 |

债券市场的信号是**矛盾但清晰的**：
- 长端利率极低（1.74%）→ 市场定价了长期的经济增长放缓甚至通缩风险
- 短端利率也低（SHIBOR 1.43%）→ 表明货币政策已大幅宽松但效果有限
- 整体而言，债券市场在定价"日本化"风险——低增长、低通胀、低利率

## 五、核心结论与风险提示

### 核心判断

1. **当前市场情绪处于极端悲观区间**。固定资产投资-15.6%与新增贷款-25.2%的双重极端信号不常出现，上一次类似情况可以追溯到2020年初疫情冲击和2015年经济调整期。

2. **三重负反馈螺旋正在形成**：信心不足→不消费不投资→收入恶化→进一步压缩信心。消费者信心89.9是一个先行指标，预示未来几个月的实体经济数据可能进一步恶化。

3. **债券市场已经在为"日本化"情景定价**。如果未来几个季度GDP增速跌破4%且通缩压力持续，当前1.74%的10年期收益率可能仍有下行空间。

4. **全球共振放大了国内悲观情绪**。科技板块一个月流出87亿美元说明海外投资者也在同步削减风险敞口，内外资同步退潮加大了市场下行压力。

### 需要关注的关键变量

- **政策应对**：未来1-2个月是否有大规模财政刺激或货币宽松出台。LPR 1年仍在3.0%，MLF利率未见更新，政策工具箱尚未充分打开。
- **信贷数据拐点**：新增贷款能否在Q3出现环比改善，是判断悲观情绪是否触顶的关键。
- **科技板块资金流向**：ETF流出是否继续加速，将决定全球风险偏好的方向。
- **房地产销售**：house_price_index 0.5%几乎持平，表明房价仍在底部徘徊，并无企稳迹象。

### 风险提示

- 以上分析基于可得数据，部分宏观数据（CPI、PPI、M2等）已超过340天未更新，存在信息滞后风险。
- -15.6%的FAI数据和-25.2%的新增贷款数据如果经后续修订有大幅调整，可能改变判断。
- 当前极端悲观情绪本身可能孕育反转——当所有人都悲观时，边际改善可能触发空头回补。

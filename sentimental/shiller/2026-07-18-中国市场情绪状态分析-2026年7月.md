# 中国市场情绪状态分析 — 2026年7月

## 概述

基于2026年7月18日可获取的多维度宏观与情绪数据，当前中国市场情绪呈现 **显著悲观但尚未极端化** 的特征。经济基本面的多项核心指标处于危险区间，但金融市场指标（债市、利率）反映的是宽松预期而非恐慌。整体上，市场处于 **"基本面承压 + 政策期待"** 的拉锯状态。

## 二、情绪综合框架分析

### 2.1 情绪-基本面矩阵

```
                    基本面
              弱             强
         ┌─────────┬─────────┐
   悲    │  当前状态│         │
   观    │  (拉锯期)│         │
情  ├─────────┼─────────┤
绪    │         │         │
   乐    │         │         │
   观    │         │         │
         └─────────┴─────────┘
```

当前落在 **"基本面弱 + 情绪悲观"** 象限，但尚未进入恐慌区域。债市定价反映的是"衰退期待宽松"的逻辑，而非信用危机定价。

### 2.2 关键矛盾的拆解

**矛盾一：PMI接近荣枯线 vs 投资断崖式下跌**

制造业PMI 50.3和非制造业PMI 50.2勉强站上荣枯线，但FAI -15.6%和新贷款 -25.2%说明经济内生动力严重不足。PMI的微弱扩张可能更多依赖政策驱动（基建、制造业升级投资）而非市场自发需求。

**矛盾二：出口暴增 vs 内需停滞**

出口+27%与零售+1%形成极端背离。可能解释：
- 全球供应链转移带来的结构性订单（中国仍是主要制造国）
- 抢出口效应（预期关税变化）
- 基数效应（去年低基数）
- **数据口径差异**——需核实出口数据是否包含价格因素

**矛盾三：低利率 + 陡峭收益率曲线**

10Y-2Y利差达48bp，曲线偏陡。在低利率环境下这是"期待复苏"的定价信号——市场预期当前宽松政策最终会推动经济回暖。但若基本面迟迟不改善，陡曲线可能转为"衰退定价"。

### 已触发的极端信号（需立即关注）

1. **FAI同比 -15.6%**：固定资产投资进入收缩区间，若无强力政策刺激，Q3可能继续恶化
2. **新增贷款同比 -25.2%**：信贷需求断崖，信用传导机制堵塞
3. **农业CGPI -4.94%**：农村/低收入群体面临通缩性收入压力，可能进一步压制消费

### 接近极端边界的信号

5. **零售增速 1.0%**：若转为负增长，将触发消费衰退警报

### 值得警惕的"假阳性"风险

6. **出口+27%/进口+36%**：与内需指标严重矛盾，高度怀疑口径/基数效应扭曲，不宜作为乐观依据

## 五、结论与展望

**核心判断：当前市场情绪处于"悲观但未恐慌"的状态，但多项指标处于极端边缘。**

- **短期（1个月内）**：市场情绪大概率维持低迷，债牛格局持续。若7月经济数据继续恶化，可能触发更大规模的政策宽松（降准降息、财政刺激）。
- **中期（1-3个月）**：FAI和信贷数据若在Q3末仍无改善，情绪可能从"悲观"滑向"恐慌"，届时股市和信用市场将承压。
- **关键监测变量**：① 7月FAI和新增贷款是否企稳 ② 消费者信心是否跌破85 ③ 政策层面是否有超预期宽松信号

**信号总评级：BEARISH | 置信度 75% | 时间跨度 1-3个月**

> ⚠️ 数据质量说明：出口/进口数据（+27%/+36%）与国内需求数据存在巨大矛盾，建议优先核实其口径与计算方式后再纳入决策依据。

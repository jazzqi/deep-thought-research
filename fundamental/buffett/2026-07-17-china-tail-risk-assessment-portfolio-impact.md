## 三、尾部风险情景分析

由于原始Agent未能生成情景，本报告基于数据独立构建以下情景：

### 情景A：中国债务-通缩陷阱（中高风险，概率40-50%）

**传导路径**：
```
PPI通缩(-3.6%) → 企业利润压缩 → 偿债能力下降
  → 银行惜贷(新贷款-25%) → 信用收缩
  → 投资冻结(FAI -15.6%) → 就业&收入下降
  → 消费疲软(社零1%) → 需求进一步萎缩
  → 价格继续下行(CPI 0%) → 循环强化
```

**对标历史**：日本1990年代"失去的十年"初期特征
- 资产价格下跌
- 信用创造机制崩溃
- 企业去杠杆
- 货币政策失效（利率已接近零）

**对投资组合影响**：
- 股票：国内消费/地产/金融板块面临20-30%下行风险
- 债券：利率债受益于宽松，但信用债违约风险上升
- 汇率：中长期贬值压力

### 情景B：房地产系统性危机升级（中风险，概率25-35%）

**触发条件**：新增贷款持续-25%意味着开发商彻底断粮。若某头部开发商违约引爆连锁反应。

**对投资组合影响**：
- 银行不良贷款激增
- 地方政府土地出让收入崩塌
- 建材/工程/家电相关产业链冲击
- 广泛信贷条件收紧

### 情景C：资本外流/汇率危机（低-中风险，概率15-20%）

**触发条件**：FDI连续-18%流出，若叠加信心恶化或地缘政治冲击，可能触发资本外逃。

**对投资组合影响**：
- CNY大幅贬值5-10%
- 外债较高的企业承压
- 输入型通胀加剧（能源CGPI已+17.74%）

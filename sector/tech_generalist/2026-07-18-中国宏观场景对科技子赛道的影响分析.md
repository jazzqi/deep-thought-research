# 中国宏观场景对科技各子赛道的差异化影响分析

**分析日期：** 2026-07-18  
**宏观场景：** Disinflationary Slowdown（低增长+低通胀+宽松政策失灵）+ Balance Sheet Recession（资产负债表衰退）  
**场景置信度：** 0.85 / 1.0  
**分析师：** tech_generalist

---

## 一、核心框架：两极撕裂的宏观格局

在深入各子赛道之前，先明确宏观环境的核心传导机制：

### 宏观传导链

```
外需繁荣（出口+27%）    内需崩塌（零售+1%，固投-15.6%）
       │                         │
       ▼                         ▼
  工业产出+5.3%          信贷-25.2%，消费信心89.9
       │                         │
       ▼                         ▼
  能源CGPI+17.7%          农产品CGPI-4.9%
       │                         │
       ▼                         ▼
 出口导向部门盈利↑        内需部门主动去杠杆
                                  │
                                  ▼
                           货币政策"推绳子"失效
                           (LPR 3.0%, SHIBOR 1.43%, 10Y国债1.74%)
```

### 对科技行业的含义

**第一层冲击：需求端撕裂**
- 外需驱动型科技子赛道：受益于全球贸易繁荣（尤其是东盟、中东、欧洲）
- 内需依赖型科技子赛道：受企业/消费者去杠杆严重拖累

**第二层冲击：融资环境宽松但传导断裂**
- 大型科技企业（国企/龙头）：融资成本极低，有资金进行AI等长期投资
- 中小科技企业（民企/初创）：信贷急剧萎缩（新增贷款-25.2%），融资环境事实上紧缩

**第三层冲击：价格信号扭曲**
- 能源成本↑（CGPI能源+17.7%）→ 高能耗科技制造（半导体制造、数据中心）成本承压
- 终端需求通缩 → 科技产品定价能力减弱，毛利收窄

## 二、科技各子赛道深度分析

### 子赛道一：半导体（Semiconductor）—— 🟡 结构性分化

#### 暴露维度
| 维度 | 影响 | 方向 |
|------|------|------|
| 出口需求 | 间接出口（嵌入电子产品）受益 | 🟢 |
| 内需 | 消费电子芯片需求疲弱 | 🔴 |
| 政策支持 | 国家自主替代战略持续加码 | 🟢 |
| 融资环境 | 龙头（中芯/华虹）政府支持，民营fabless融资恶化 | 🟡 |
| 成本 | 电力/能源成本上升侵蚀制造端利润 | 🔴 |
| 地缘风险 | 美国制裁持续，设备进口受限 | 🔴 |

#### 关键判断

**制造端（Foundry）：** 短期内需疲弱导致产能利用率承压。但国家"芯片自主"战略在资产负债表衰退时期反而可能加大财政支持力度——因为刺激基建/地产已无效，半导体投资是少数兼具"国家安全属性"和"产业拉动效应"的方向。从新闻看到壁仞科技推出1024卡超节点方案、NPO光互连等，表明国产算力在制裁下仍在加速追赶。

**设计端（Fabless）：** 严重分化。面向出口市场（IoT、家电MCU、工业控制）和AI加速芯片的公司相对有支撑；面向国内消费电子（手机/平板CIS、显示驱动）的公司面临终端需求崩塌式收缩。

**设备/材料：** 国产替代逻辑最强。受制于制裁但也是政策投入最集中的领域。不过需注意：在资产负债表衰退环境下，设备投资周期可能因下游Fabless融资困难而拉长。

#### 投资含义
- **看多：** 半导体设备/材料国产替代龙头、AI算力芯片
- **看空：** 消费电子芯片设计、低端MCU
- **中性但谨慎：** 晶圆代工（产能利用率存疑）

### 子赛道二：人工智能 / 大模型（AI/LLM）—— 🟢 结构性看多

#### 暴露维度
| 维度 | 影响 | 方向 |
|------|------|------|
| 出口需求 | AI模型/能力出海（印企依赖中国模型） | 🟢 |
| 内需（企业） | IT预算收缩，但AI是少数"必投"方向 | 🟡 |
| 内需（消费者） | C端AI产品付费意愿弱 | 🔴 |
| 政策支持 | 国家强推，WAIC高规格举办 | 🟢 |
| 融资 | 头部集中，长尾AI初创融资困难 | 🟡 |
| 成本 | 算力成本高（电力+GPU进口受限） | 🔴 |

#### 关键判断

**AI基建层（算力/云）：** 受益于政策驱动和头部企业（阿里、百度、华为）的持续投入。阿里巴巴刚发布企业级AI应用创作平台"秒悟团队版"；国家能源局宣布加快构建新型能源体系"赋能AI发展"，表明AI算力已被提升到国家战略高度。但能源成本上升（CGPI能源+17.7%）对数据中心运营构成压力。

**AI模型层：** 百模大战进入淘汰期。日媒报道"印企越来越依赖中国AI大模型"，说明中国AI模型已在全球南方市场建立出口能力。国内竞争激烈但头部效应明显。

**AI应用层：** 典型的B2B/SaaS商业模式在内需疲弱环境下面临挑战——企业客户IT预算收缩、决策周期拉长。滴滴在WAIC分享AI升级出行体验，但释放"服务消费新潜力"恰恰说明消费端应用的货币化困难。

#### 投资含义
- **看多：** AI算力基建（服务器/光模块/液冷）、头部云厂商、AI出海能力
- **中性：** AI应用/SaaS（等待企业IT支出拐点）
- **看空：** 烧钱式AI初创（融资环境恶化）

### 子赛道三：消费电子（Consumer Electronics）—— 🔴 看空

#### 暴露维度
| 维度 | 影响 | 方向 |
|------|------|------|
| 出口 | 苹果供应链仍有支撑 | 🟡 |
| 内需 | 零售+1%，消费者信心89.9 → 换机周期拉长 | 🔴 |
| 价格 | 通缩环境 → 产品ASP承压 | 🔴 |
| 库存 | 渠道库存高企 | 🔴 |

#### 关键判断

这是科技行业中受资产负债表衰退冲击最直接的子赛道。消费者信心处于89.9的极度低迷水平，零售总额几乎零增长，手机/PC/平板等可选消费电子产品首当其冲。

苹果在日本市场涨价10%（iPhone）反映了全球通胀压力，但在中国市场苹果面临"涨价→需求进一步萎缩"的困境。安卓阵营则面临更严峻的国产替代内卷——华为回归进一步压缩了其他品牌的生存空间。

可穿戴设备、IoT消费产品等同样受制于消费降级趋势。

#### 投资含义
- **全面看空：** 国内消费电子品牌、手机供应链（内需暴露高者）
- **谨慎看多：** 苹果核心供应链（但需关注中美脱钩风险）
- **规避：** 高库存消费电子渠道商

### 子赛道四：新能源汽车 / 锂电池（EV & Battery）—— 🟡 结构性分化

#### 暴露维度
| 维度 | 影响 | 方向 |
|------|------|------|
| 出口 | 锂电池/电动车出口持续高增 | 🟢 |
| 内需 | 国内EV渗透率已高，增量放缓 | 🟡 |
| 价格 | 价格战持续，行业利润率承压 | 🔴 |
| 政策 | 锂电池消费税调整，利好整车企业自造电池 | 🟢 |
| 融资 | 产能过剩行业融资收紧 | 🔴 |

#### 关键判断

崔东树指出"电池企业竞争格局形成宁德时代和比亚迪两者相对较强的特征"，这印证了行业分化。锂电池消费税调整"对整车企业造电池是有力助推"——这意味着：

1. **龙头集中：** 宁德时代、比亚迪凭借规模和成本优势继续巩固地位
2. **出口驱动：** 锂电池出口受益于全球能源转型需求，与宏观"外需繁荣"逻辑一致
3. **内需放缓：** 国内EV渗透率已达高位，增量空间收窄，价格战侵蚀利润
4. **尾部出清：** 二三线电池厂和造车新势力面临融资困难

#### 投资含义
- **看多：** 动力电池双寡头（宁德时代、比亚迪）、锂电材料出口龙头
- **中性：** 整车（价格战持续，盈利不确定）
- **看空：** 二三线电池厂、烧钱造车新势力

### 子赛道五：企业服务 / SaaS（Enterprise SaaS）—— 🔴 看空

#### 暴露维度
| 维度 | 影响 | 方向 |
|------|------|------|
| 内需（企业IT支出） | 企业去杠杆 → IT预算收缩 | 🔴 |
| 客户付款能力 | 中小企业现金流紧张 → SaaS续费率下降 | 🔴 |
| 融资 | VC/PE收缩 → SaaS初创融资困难 | 🔴 |
| 政策 | 信创替代有一定支撑 | 🟡 |
| 成本 | 人力成本刚性 | 🔴 |

#### 关键判断

这是资产负债表衰退中受影响最严重的科技子赛道之一。原因：

1. **企业主动去杠杆：** 新增贷款-25.2%，企业优先偿还债务而非投资IT
2. **SaaS订阅模式的脆弱性：** 在经济下行期，SaaS订阅是企业最容易削减的"非核心"支出
3. **中小企业出清：** SaaS的主要客户群（中小企业）在信贷紧缩中加速倒闭
4. **长尾SaaS公司融资断裂：** VC投资大幅萎缩，大量SaaS初创面临断粮

唯一的结构性支撑来自信创（国产替代），政府/国企的信创需求相对刚性，但规模不足以对冲整个市场的萎缩。

#### 投资含义
- **全面看空：** 通用型SaaS（CRM/ERP/协作）、面向中小企业的SaaS
- **谨慎看多：** 信创SaaS（政府/国企客户）、与AI结合的高附加值工具
- **规避：** 尚未盈利的SaaS公司

### 子赛道六：互联网平台（Internet Platforms）—— 🟡 结构性分化

#### 暴露维度
| 维度 | 影响 | 方向 |
|------|------|------|
| 广告收入 | 企业缩减营销预算 → 广告承压 | 🔴 |
| 电商GMV | 消费降级 → 客单价下降，GMV增速放缓 | 🔴 |
| 云/AI | 阿里云等新增长引擎仍在增长 | 🟢 |
| 海外业务 | Temu/Shein/TikTok等出海强劲 | 🟢 |
| 游戏 | 可选消费属性，但海外发行有支撑 | 🟡 |
| 监管 | 监管环境趋于稳定 | 🟡 |

#### 关键判断

互联网平台是"内外需分化"最典型的科技子赛道：

**国内业务（承压）：**
- 广告：企业缩减营销预算（零售+1%环境下，广告主预算紧张）
- 电商：GMV增速放缓，客单价下降（消费降级）
- 本地生活：到店/到家消费受消费者信心低迷拖累

**海外业务（支撑）：**
- 出海电商（Temu/Shein/TikTok Shop）受益于全球消费
- 游戏出海（米哈游、网易等）持续增长
- 工具类APP全球下载量稳定

#### 投资含义
- **看多：** 出海业务占比高的平台、云/AI业务领先的平台
- **中性：** 收入结构均衡的平台（防守能力强，但缺乏上行催化）
- **看空：** 严重依赖国内广告/电商收入的平台

### 子赛道七：通信 / 算力基础设施（Telecom & Computing Infrastructure）—— 🟡 中性偏多

#### 暴露维度
| 维度 | 影响 | 方向 |
|------|------|------|
| 5G建设 | 高峰期已过，投资放缓 | 🔴 |
| AI数据中心 | AI算力需求爆发式增长 | 🟢 |
| 光模块/光互连 | 出口+AI双驱动 | 🟢 |
| 融资 | 大型国企，融资无虞 | 🟢 |
| 成本 | 电力成本上升 | 🔴 |

#### 关键判断

5G投资高峰期已过，运营商的资本开支在收缩。但AI数据中心建设正在接力成为新的增长引擎。壁仞科技推出NPO光互连、分布式解耦架构超节点方案（最大1024卡），说明算力基础设施（光模块、液冷、高速互联）正处于高速发展期。

运营商（移动/电信/联通）受益于：
- 低利率环境降低财务成本（SHIBOR 1.43%）
- AI云业务增长
- 高股息防御属性（在当前环境下有吸引力）

#### 投资含义
- **看多：** 光模块/光互连（AI+出口双驱动）、液冷解决方案、AI服务器
- **中性看多：** 电信运营商（高股息+低利率防御）
- **中性：** 传统通信设备（5G投资放缓）

#### 投资含义

## 四、跨子赛道的协同效应与风险传导

### 正向协同

1. **AI → 算力基础设施：** AI大模型训练和推理需求直接拉动光模块、服务器、液冷需求，这是当前最确定的增长链
2. **半导体设备 → 晶圆代工：** 国产替代推动设备采购，但需注意代工厂产能利用率下行风险
3. **出口繁荣 → 跨境电商/物流科技：** 外贸高增拉动跨境电商SaaS、跨境支付、物流科技
4. **AI → 半导体设计：** AI推理芯片（NPU/ASIC）需求增长，但面向手机/AIoT的AI芯片受消费电子拖累

### 负向传导链

1. **消费电子崩塌 → 半导体设计 → 晶圆代工：** 消费电子芯片需求下滑向上游传导，可能导致代工厂产能利用率下降
2. **企业去杠杆 → SaaS → 云服务：** 企业IT预算收缩不仅影响SaaS，也间接影响云服务增速
3. **消费降级 → 互联网广告 → 中小互联网公司：** 广告收入下降进一步压缩中小互联网公司研发投入

### 关键风险传导路径

```
贸易摩擦升级
    ↓
出口引擎熄火（当前唯一增长动力）
    ↓
工业产出增速回落 → 科技制造（半导体代工/消费电子代工）受冲击
    ↓
内需进一步恶化 → 所有内需依赖型科技子赛道（SaaS/广告/消费电子）加速下挫
    ↓
全面衰退风险
```

## 五、与宏观场景的交叉验证

### 验证点1：资产负债表衰退对科技行业的特殊影响

科技行业通常被认为具有"逆周期"属性（数字化转型在经济下行期反而加速），但本次衰退的**资产负债表属性**改变了这一规律：

- **传统逆周期逻辑：** 经济不好 → 企业寻求降本增效 → 加大IT/SaaS投入
- **资产负债表衰退逻辑：** 企业负债率过高 → 优先偿债而非投资 → 削减一切非必要支出 → IT/SaaS首当其冲

因此，**资产负债表衰退下的科技投资逻辑不同于普通经济周期**。只有两类科技支出具有刚性：
1. **能直接带来收入增长的支出**（如AI驱动的销售工具、电商平台）
2. **政策强制要求的支出**（如信创、数据安全）

### 验证点2：货币政策失效对科技融资的影响

LPR 3.0%和SHIBOR 1.43%的极低利率并未转化为科技企业的融资便利：
- 银行风险偏好下降 → 对科技企业（尤其是民企）惜贷
- 新增贷款-25.2% → 科技企业融资量萎缩
- 这意味着**科技行业的"融资驱动型"子赛道（AI初创、SaaS、半导体设计）面临实质性紧缩**，与利率水平严重背离

### 验证点3：PPI/CPI分化对科技利润的影响

PPI 4.1%（成本推动）叠加CPI仅1.0%（需求疲弱），意味着：
- 上游（原材料、能源）成本上升
- 下游（终端产品、服务）价格无法传导
- 科技制造企业的**毛利率将持续承压**
- 这在消费电子、光伏、新能源汽车中表现尤为明显

## 七、关键结论

1. **极端分化是核心主题：** 在中国资产负债表衰退+Disinflationary Slowdown场景下，科技行业不存在"整体性机会"，只有结构性分化。出口驱动型（AI算力、电池、半导体设备）受益，内需依赖型（SaaS、消费电子、广告）承压。

2. **资产负债表衰退改变了科技投资的传统逻辑：** 科技行业"逆周期"属性在此次衰退中失效。企业优先偿债而非数字化转型，导致SaaS/IT服务需求收缩。

3. **货币政策传导断裂是科技融资的最大风险：** 低利率（LPR 3.0%）并未传导至科技企业融资，新增贷款-25.2%意味着大量科技初创面临断粮风险。

4. **AI是唯一的"确定性"增长叙事：** 受益于国家战略推动+全球AI需求+出口能力，AI算力基础设施是当前确定性最高的科技配置方向。

5. **最大尾部风险：** 贸易摩擦导致出口引擎熄火。当前中国科技行业的唯一增长动力是外需，若出口增速显著回落（从+27%降至个位数），所有出口导向型子赛道将同步承压，届时全行业面临系统性风险。

6. **7月PMI是近期最重要的观测点：** 若PMI跌破50，需立即增加防御配置；若财政刺激超预期，可战术性加配内需敏感型科技板块。

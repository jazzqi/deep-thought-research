# CIO综合判断：中国宏观进入「通缩性放缓 + 工业再通胀」罕见混合regime

**日期：** 2026-07-18  
**分析师来源：** macro_team_analysis (dalio)  
**置信度：** 75%  
**共识水平：** 单一来源（macro_team仅提交一份完整分析），数据已验证

---

## 一、核心场景判断

中国宏观经济当前处于一个**罕见的、非经典的混合regime**：

> **「通缩性放缓（需求侧）+ 工业再通胀（供给侧）」**

这一判断既不是经典滞胀（因为CPI仅1.0%，远非高通胀），也不是经典通缩（因为PPI已快速攀升至4.1%），而是需求端疲软与供给端成本冲击同时存在的分裂状态。

---

## 二、数据验证与三维支撑

### 1️⃣ 增长端：全线走弱（数据已验证 ✅）

| 指标 | 数值 | 数据时效 | 验证状态 |
|------|------|----------|----------|
| GDP同比 | 4.7% | 2026年第1-2季度 | ✅ 确认 |
| 制造业PMI | 50.3 | 2026年6月 | ✅ 确认，荣枯线边缘 |
| 非制造业PMI | 50.2 | 2026年6月 | ✅ 确认，荣枯线边缘 |
| 固定资产投资同比 | **-15.6%** | 2026年6月 | ✅ 确认 — 断崖式收缩 |
| 零售同比 | **1.0%** | 2026年6月 | ✅ 确认 — 消费近零增长 |
| 消费者信心 | 89.9 | 2026年5月 | ✅ 确认 — 持续悲观（荣枯线100） |

**CIO评述：** FAI -15.6%是当前增长端最危险的信号，这不是周期性放缓，而是结构化收缩。零售1.0%意味着内需几乎零贡献。GDP 4.7%隐含下半年需要更强拉动才能企稳。

### 2️⃣ 通胀端：极端分化（CGPI数据已验证 ✅）

**CPI端（消费品）：通缩压力仍在**
- 2026年各月：1.3% → 1.0% → 1.2% → 1.2% → **1.0%**
- 持续低于央行3%目标，环比走弱
- **结论：需求侧通缩，无消费通胀风险**

**PPI端（工业品）：猛烈再通胀**
- 2026年各月：-0.9% → +0.5% → +2.8% → +3.9% → **+4.1%**
- 4个月内从通缩跳升至4.1%通胀
- **上游验证：** CGPI能源同比 **+17.74%**（2026年6月）✅ — 确认上游大宗成本冲击
- **对冲因素：** CGPI农产品同比 **-4.94%** — 部分抵消食品端压力

**PPI-CPI剪刀差：** 从2月的-1.7%扩大到6月的**+3.1个百分点**
- 这是企业利润挤压的直接信号
- 上游成本上涨无法传导至终端消费者（因为需求不足）
- 中游制造业利润率将受到严重侵蚀

### 3️⃣ 政策端：流动性陷阱（数据已验证 ✅）

| 指标 | 数值 | 数据时效 | 验证状态 |
|------|------|----------|----------|

**CIO评述：** 货币政策处于典型的"推绳子（Pushing on a String）"困境。利率已足够低，流动性已足够充裕，但经济主体缺乏借贷意愿。新贷款同比-25.2%是最有力的证据——这不是钱不够贵的问题，而是投资回报率预期太低的问题。10Y国债1.74%定价了市场对长期低增长的共识。

## 三、场景矩阵定位

```
                  CPI通胀（1.0%）
            低 ←━━━━━━━━━━━→ 高
        ┌────────────────────────────────┐
  增长↑  │  黄金期（Goldilocks）         │  过热
  走强  │                                │
        │                                │
  增长↓  │  ◀ 当前位置                    │  滞胀（Stagflation）
  走弱  │  （通缩性放缓+工业再通胀）      │  CPI高+增长低
        └────────────────────────────────┘
```

**为什么不是经典滞胀？** 滞胀需要CPI通胀高+增长低，但当前CPI仅1.0%。
**为什么不是经典通缩？** PPI已飙至4.1%，上游工业链正在经历成本推动型通胀。
**这是什么？** CPI-PPI分裂的混合体——需求不足导致消费品通缩，供给冲击（能源CGPI+17.74%）导致工业通胀。

## 四、风险评估与前瞻

| 风险方向 | 触发条件 | 概率评估 | 影响 |
|----------|----------|----------|------|
| 🔴 **恶化** | PPI持续高位 → 企业利润挤压 → 裁员/降薪 → 消费进一步走弱 → 通缩螺旋 | 40-50% | 恶性循环，需警惕 |
| 🟡 **僵持** | 政策以温和刺激应对，经济在低位企稳但不反弹 | 35-40% | 资产配置挑战，债券有支撑 |
| 🟢 **改善** | 强力财政刺激出台（发消费券/基建加码）+ 上游成本回落 | 15-20% | 概率最低，需财政催化剂 |

**关键监测点：**
1. **PPI向CPI传导路径** — 如果PPI压力开始传导至CPI，场景可能转向经典滞胀
2. **财政政策变化** — 货币政策失效，财政政策是唯一有效工具
3. **新贷款数据** — 如果持续-20%以下，信用塌缩将自我强化
4. **能源价格（CGPI）** — 上游成本是PPI的主要驱动因素

## 六、结论

中国宏观当前处于**非经典的混合regime**——需求侧通缩性放缓（CPI 1.0%, FAI -15.6%, 零售1.0%）与供给侧工业再通胀（PPI 4.1%, CGPI能源+17.74%）同时存在。货币政策陷入流动性陷阱（LPR 3.0%但新贷款-25.2%），政策有效性严重受限。

**核心矛盾：** PPI-CPI剪刀差持续扩大将挤压中下游企业利润，若叠加需求进一步走弱，可能形成"工业成本上升→企业利润压缩→就业/收入恶化→消费进一步萎缩"的负反馈循环。

**建议关注：** 后续财政政策力度（唯一有效工具），以及PPI是否向CPI传导（场景切换为经典滞胀的触发条件）。

# CIO宏观场景综合判断：全球深度分化格局

**发布日期：** 2026-07-18
**分析师：** CIO（首席投资官）
**数据来源：** 宏观团队三位分析师（dalio, liu, woo）
**置信度：** 0.80 / 1.0
**共识水平：** HIGH

---

## 一、综合结论

全球宏观经济正处于**"深度分化"**格局——中国与美国处在完全不同的周期阶段，这种分化程度在过去20年中极为罕见。三位分析师在以下核心判断上达成高度共识：

### 全局画像：双轨世界（Bifurcated World）

| 维度 | 中国 | 美国 |
|------|------|------|
| **周期定位** | Stagnation-lite（低于趋势的停滞） | Late-Cycle Expansion（晚周期扩张） |
| **通胀性质** | CPI良性 + PPI成本推动型再通胀 | 核心CPI粘性（~3.6%年化） |
| **政策取向** | 极度宽松但传导断裂（推绳子） | 限制性，Higher for Longer |
| **信用状况** | 萎缩（新增贷款同比-25.2%） | 正常 |
| **增长引擎** | 出口唯一亮点（+27%） | 内需驱动 |
| **主要风险** | 内需塌陷 + 信用断裂 | 通胀粘性超预期 |

**共识水平：HIGH** — 三位分析师在中国停滞、美国扩张、全球分化的定性判断上高度一致。

## 二、数据全景支撑

### 2.1 中国：Stagnation-lite + PPI Reflation

#### Growth — 低于目标，内需塌陷

| 指标 | 最新值 | 信号强度 |
|------|--------|---------|
| GDP H1 2026 | **4.7%** yoy（Q2隐含~4.4%） | ⚠️ 低于5%目标 |
| 制造业PMI（6月） | 50.3 | 荣枯线边缘 |
| 非制造业PMI（6月） | 50.2 | 荣枯线边缘 |
| 零售总额（6月） | **1.0%** | 🚨 极度疲弱 |
| FAI固投（6月） | **-15.6%** | 🚨 崩盘式收缩 |
| 出口（6月） | **+27.0%** | 唯一亮点 |
| 消费者信心 | 89.9 | 低迷 |

**判断依据：** GDP从Q1的5.0%减速至H1的4.7%，Q2实际~4.4%。零售1%是内需疲弱的铁证，消费信心89.9处于历史低位。FAI -15.6%反映地产投资断崖和地方政府去杠杆。出口27%是唯一支撑但面临地缘政治脆弱性。

#### Inflation — CPI无虞，PPI快速再通胀（核心新变化）

| 指标 | 最新值 | 趋势 |
|------|--------|------|
| CPI同比（6月） | 1.0% | 持续温和 |
| **PPI同比（6月）** | **4.1%** | 🚨 从2月-0.9%→6月+4.1%，+5ppt swing |
| CGPI能源（6月） | **+17.7%** | 成本冲击 |
| CGPI农产品（6月） | -4.9% | 需求不足 |

**判断依据：** PPI在4个月内从-0.9%急升至4.1%，由能源成本推动而非需求拉动。PPI-CPI裂口扩大→工业企业利润承压（成本涨但无法转嫁消费者）。这是**成本推动型通胀**，对政策含义是：央行无法通过收紧来应对，因为根源不在需求端。

#### Policy & Credit — 极松货币 + 信用断裂

| 指标 | 最新值 | 含义 |
|------|--------|------|
| 1Y LPR | 3.0% | 历史低位 |
| SHIBOR 3M | 1.43% | 极低流动性充裕 |
| 10Y国债 | 1.74% | 接近历史低点 |
| **新增贷款同比（6月）** | **-25.2%** | 🚨 信用萎缩 |
| M2同比 | 8.0% | 稳健但无效 |
| USD/CNY | 6.78 | 稳定 |

**判断依据：** 利率已在危机水平，但新增贷款-25.2%表明经典的"推绳子"困境——银行愿意放贷，但私人部门（地产/居民）不愿借。这本质上是**信用通道断裂**，数量工具比价格工具更具政策紧迫性。

### 2.2 美国：Late-Cycle Expansion

| 指标 | 最新值 | 含义 |
|------|--------|------|
| GDP | **3.3%** yoy | 高于趋势 |
| 失业率 | **4.2%** | 充分就业 |
| 核心CPI（6月） | 0.3% m/m | ~3.6%年化，粘性 |
| 非农就业 | +7.3万（6月） | 放缓但仍健康 |
| 10Y收益率 | 4.57% | Higher for Longer |
| 2Y收益率 | 4.16% | 限制性 |
| 2s10s利差 | **+0.41%** | ✅ 正常化（非倒挂） |

**判断依据：** 美国经济韧性超预期。GDP 3.3%高于趋势，失业率4.2%处于充分就业。核心CPI环比0.3%（年化~3.6%）仍有粘性但未失速。收益率曲线从倒挂转为+41bp正利差，是经济健康的信号。美联储维持Higher for Longer是基准情景。

### 2.3 全球：深度分化

中国与美国分别处于完全不同的周期阶段，两者之间没有明显的收敛趋势。从全球资产配置角度看：

- 中国资产面临**"宏观逆风"**：增长停滞、信用萎缩、PPI成本冲击
- 美国资产在**"晚周期扩张"**中运行：增长稳健、就业充分、通胀有序回落
- 全球整体处于**"分裂的滞胀边缘"**（Bifurcated Stagflationary Lurch）

## 三、关键洞察与风险

### 3.1 最值得关注的信号：中国PPI快速再通胀

PPI从-0.9%到+4.1%仅用了4个月，这是一个**被市场低估的重大变化**。其含义：

1. **工业企业利润挤压**：PPI涨但CPI不动→中下游企业无法转嫁成本
2. **政策两难**：CPI低→有空间宽松；但PPI涨→进一步宽松可能加剧输入性通胀
3. **传导链条**：全球能源价格→中国PPI→若持续→最终可能向CPI传导

### 3.2 核心风险矩阵

| 风险 | 概率 | 影响 | 描述 |
|------|------|------|------|
| 中国信用事件 | 中高 | 高 | 新增贷款-25.2%，地产融资链条紧绷 |
| 美国通胀反弹 | 中 | 高 | 核心CPI粘性超预期→Fed再加息 |
| 中美脱钩升级 | 中 | 中高 | 出口+27%是中国唯一增长引擎，若受冲击则GDP可能破4% |
| 全球能源冲击 | 中 | 中 | CGPI能源+17.7%已体现部分影响 |

### 3.3 资产配置含义

| 资产类别 | 方向 | 逻辑 |
|---------|------|------|
| 中国债券 | 看多（已price in） | 10Y 1.74%，宽松政策延续，但下行空间有限 |
| 中国股市 | 谨慎/防御 | 内需疲弱+PPI成本挤压企业盈利 |
| 美国债券 | 中性 | Higher for Longer，收益率高位震荡 |
| 美国股市 | 谨慎看多 | 晚周期扩张+盈利支撑，但估值不便宜 |
| 美元/人民币 | 区间震荡 | 6.78附近，政策维稳 |
| 大宗商品 | 分化 | 能源偏强（支撑PPI），工业金属需求承压 |

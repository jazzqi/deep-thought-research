## 一、核心宏观场景：全球"双约束"格局

### 1.1 中国：通缩性放缓 + 工业再通胀 = 罕见混合regime

中国宏观经济当前处于**非经典的混合regime**——需求侧通缩性放缓与供给侧工业再通胀同时存在。这不是经典滞胀（CPI仅温和），也不是经典通缩（PPI已飙升），而是CPI-PPI严重分裂的罕见状态。

| 维度 | 指标 | 最新值 | 数据日期 | 含义 |
|------|------|--------|----------|------|
| **增长** | GDP同比 | 4.7% | 2026H1 | 较2025年5.0%明显放缓 |
| | 制造业PMI | 50.3 | 2026年6月 | 荣枯线边缘，实质零增长 |
| | 非制造业PMI | 50.2 | 2026年6月 | 服务业同样疲弱 |
| **需求** | 零售同比 | +1.0% | 2026年6月 | 近零增长，内需严重不足 |
| | **固定资产投资同比** | **-15.6%** | 2026年6月 | **断崖式收缩，最危险信号** |
| | 消费者信心 | 89.9 | 2026年5月 | 持续悲观（荣枯线100） |
| **信用** | LPR 1Y | 3.0% | 最新 | 历史低位 |
| | SHIBOR 3M | 1.43% | 2026年7月17日 | 极度宽松 |
| | 10Y国债 | 1.7404% | 2026年7月17日 | 全球极低，定价长期低增长 |
| | **新房贷款同比** | **-25.2%** | 2026年6月 | **流动性陷阱铁证——信用塌缩** |
| **成本** | **能源CGPI同比** | **+17.74%** | 2026年6月 | **成本冲击源头确认** |
| | CGPI农产品同比 | -4.94% | 2026年6月 | 部分抵消食品端压力 |

**货币政策困境**：完全处于"推绳子（Pushing on a String）"状态——不是钱不够便宜，而是投资回报率预期太低，经济主体不愿借贷。新房贷款-25.2%是这一困境的铁证。

**核心矛盾**：PPI-CPI剪刀差扩大（能源CGPI+17.74% vs CPI温和）挤压中下游利润。若需求进一步走弱，可能触发"成本上升→利润压缩→就业恶化→消费萎缩"的负反馈循环。

### 1.2 美国：美联储"谨慎观望"——通胀粘性 vs 就业走弱

美联储处于双重约束中，既不能降息也不能加息：

| 抑制降息的因素 | 阻止加息的因素 |
|---|---|
| CPI同比3.5%（远高于2%目标） | 失业率升至4.2%（从周期低点持续回升） |
| Core CPI MoM 0.3%（折年3.6%，粘性显著） | 非农仅7.3万（明显疲软） |
| GDP 3.3%（仍高于趋势） | 工业产出-0.1%（制造业收缩） |

**关键Gap**：市场定价了1-2次降息（2Y=4.16%），而Fed点阵图暗示"年内或不降息"，Gap约25-50bp。市场低估了通胀粘性，高估了降息概率。

**反身性风险**：降息预期的自我否定循环——市场定价降息→金融条件宽松→刺激需求→通胀更难降→Fed无法降息→预期落空。

## 二、对主要行业的影响评估

### 2.1 防御性结构性机会（超配 Overweight）

#### 🟢 AI / 大模型 —— 最优先超配
| 维度 | 判断 |
|------|------|
| 宏观敏感度 | 低偏中 |
| 配置建议 | **超配（Overweight）** |

**逻辑链条**：
- ✅ **国家级战略**：AI是中国科技自立的核心方向，算力基建不受宏观波动影响
- ✅ **降本增效刚需**：经济下行期，企业用AI替代人工的需求反而上升
- ✅ 开源生态活跃（Grok Build 18.6k stars，Claude Code IDE等）
- ⚠️ 注意：$20/月定价天花板可能限制商业化空间

#### 🟢 网络安全 —— 防御性刚性配置
| 维度 | 判断 |
|------|------|
| 宏观敏感度 | 中低 |
| 配置建议 | **超配（Overweight）** |

**逻辑链条**：
- ✅ 威胁不随经济减少（CVE-2026-63030 WordPress核心RCE漏洞持续爆发）

- ✅ AI安全增量（对抗攻击、模型安全、数据安全）
- ⚠️ 企业IT预算整体收缩时可能小幅受压，但弹性远低于其他IT支出

#### 🟢 机器人 / 自动化 —— 结构性顺风

- ✅ 技术突破加速（RoboTTT论文等）
- ✅ 不受消费端疲弱直接影响，属于企业端降本决策

## 三、情景分析与概率权重

### 中国宏观情景

| 情景 | 概率 | 触发条件 | 对配置的影响 |
|------|------|----------|-------------|
| 🔴 **恶化（通缩螺旋）** | 40-50% | PPI高位挤压利润→裁员→消费再走弱→企业违约上升 | 全面转向防御：利率债+黄金+网络安全+AI |
| 🟡 **僵持** | 35-40% | 温和刺激，低位企稳不反弹 | 维持当前配置：超配结构性防御，低配周期 |
| 🟢 **改善** | 15-20% | 强力财政刺激+上游成本回落+信心修复 | 适度增配周期和消费，但需确认信号 |

### 美联储情景

| 情景 | 概率 | 触发条件 | 对配置的影响 |
|------|------|----------|-------------|
| 🔴 **通胀粘性+Fed不降息** | 45% | 核心CPI连续维持0.3%以上 | 美债利率维持高位，利空风险资产 |
| 🟡 **软着陆+适度降息** | 35% | 通胀稳步回落至2.5% | 利好全球风险资产 |
| 🟢 **衰退+被迫降息** | 20% | 非农连续低于5万，失业率突破4.5% | 避险为王，美债大涨 |

## 四、CIO战略判断总结

### 总体框架

在当前"中国通缩性放缓+工业再通胀"混合regime与"美联储谨慎观望"的全球双约束格局下，**核心策略是从"贝塔驱动"转向"阿尔法防御"**——放弃对宏观周期反转的押注，聚焦于结构性确定性的方向。

### 行业暴露调整建议

| 方向 | 行业 | 调整建议 | 核心逻辑 |
|------|------|----------|----------|
| **🟢 超配** | AI / 算力基础设施 | +2% → 5% | 国家级战略+降本增效刚需 |
| **🟢 超配** | 网络安全 | +1% → 4% | 防御性刚性需求+合规驱动 |
| **🟢 超配** | 机器人/自动化 | +1% → 3% | 机器换人结构性顺风 |
| **🟢 超配** | 中国利率债 | 维持现状仓位 | 增长放缓+宽松货币 |
| **🟡 中性** | 半导体（国产替代） | 维持 | 需区分方向 |
| **🟡 中性** | 新能源 | 维持 | 长期趋势不变但短期承压 |
| **🔴 低配** | A股周期行业 | -3% | FAI断崖+PPI-CPI剪刀差 |
| **🔴 低配** | 消费/零售 | -2% | 零售+1%+消费者信心89.9 |
| **🔴 低配** | 房地产及地产链 | -3% | 结构性出清未完成 |
| **🔴 低配** | 中下游制造 | -2% | 利润率挤压+需求走弱 |
| **🔴 低配** | 信用债 | -1% | 利润率挤压→信用风险上升 |

### 关键待办事项

1. **密切跟踪财政政策信号**——这是当前唯一有效工具。关注特别国债发行、地方专项债加速、消费补贴等信号
2. **监控PPI-CPI剪刀差变化**——如果PPI开始回落（上游成本缓解）或CPI回升（需求改善），将是regime切换的关键信号
3. **等待美联储数据确认**——不参与当前过度拥挤的降息交易，耐心等待通胀数据给出清晰信号
4. **关注能源成本走势**——CGPI能源+17.74%是当前中下游利润挤压的核心源头，油价和能源商品价格走势至关重要

## 一、中国：低增长 + 成本推动型通胀 + 政策传导失灵

### 共识数据基础

| 指标 | 数值 | 信号 |
|------|------|------|
| GDP同比（H1 2026） | 4.7% | 低于5%目标，减速趋势 |
| 制造业PMI | 50.3 | 刚过荣枯线，动能趋零 |
| 非制造业PMI | 50.2 | 同上 |
| 社零同比 | +1.0% | 消费极度疲弱 |
| 固投同比 | -15.6% | 大幅收缩 |
| 新增贷款同比 | -25.2% | 信用塌陷 |
| 消费者信心 | 89.9 | 低于荣枯线 |
| 10Y国债收益率 | 1.74% | 历史低位，定价通缩 |
| SHIBOR 3M | 1.43% | 极致宽松 |
| CGPI能源同比 | +17.7% | 成本端显著上涨 |
| 出口同比 | +27% | 外需强劲（"外热内冷"核心证据） |

### 分析师标签对照

| 分析师 | 标签 | 强调重点 |
|--------|------|----------|
| **Dalio** | **Stagflation-lite（类滞胀）** | 低增长+PPI快速上行+内需通缩的混合体 |
| **Zhou Jintao** | **Disinflationary Slowdown + Policy Impotence（通缩性放缓+政策失效）** | 信用塌陷、推绳子效应、货币政策传导断裂 |

### CIO统一解读

**两个标签描述的是同一硬币的两面：**

- **Dalio的"类滞胀"视角**侧重PPI端——能源CGPI +17.7%带动的成本推动型通胀正在挤压中下游制造业利润（CPI-PPI剪刀差转负），这是供给侧的"滞胀"压力。
- **Zhou的"通缩性放缓"视角**侧重需求端——CPI仅约1.0%、消费信心低迷、信用需求崩塌，这是需求侧的"通缩"压力。
- **统一判断：这既不是经典滞胀也不是纯通缩，而是中国特有的"外热内冷+上游通胀下游通缩"结构分裂。**

**核心机制链条：**
出口强劲（+27%）→ 上游能源/原材料涨价 → PPI上行 → 制造业成本上升 → 但内需极弱（零售+1%、固投-15.6%）→ 无法向CPI传导 → 中下游利润被挤压 → 企业不愿投资/居民不愿消费 → 信贷需求崩塌（新贷-25.2%）→ 宽货币但紧信用 → 政策"推绳子"

**关键分歧点：无。** 两位分析师的标签差异是语义重点不同，而非判断分歧。

## 二、美国：Late-cycle Stickyflation

### 共识数据基础

| 指标 | 数值 | 信号 |
|------|------|------|
| GDP同比 | 3.3% | 增长稳健，非过热 |
| Core CPI月环比 | 0.3%（折年3.6%） | 回落停滞，远高于2%目标 |
| 失业率 | 4.2% | 接近充分就业 |
| 10Y收益率 | 4.57% | 限制性水平 |
| 2s10s利差 | +0.41% | 曲线正常化（不再倒挂） |

### CIO统一解读

**两位分析师完全一致。** 美国处于典型的late-cycle阶段——增长有韧性但通胀粘性超预期，美联储面临"维持限制性利率更久"的困境。曲线正常化本身是经济尚未衰退的信号，但通胀无法回到2%意味着降息空间极为有限。

## 三、全球：Divergent Regime（双轨分化）

### 统一框架

```
                    ┌─────────────────────────────────────┐
                    │         全球宏观双轨分化               │
                    ├──────────────────┬──────────────────┤
                    │      中国         │       美国        │
├──────────────────┼──────────────────┼──────────────────┤
│ 增长周期         │ 低于趋势（4.7%）   │ 稳健（3.3%）       │
│ 通胀方向         │ 结构性分裂         │ 粘性偏高           │
│ 政策取向         │ 宽松但失效         │ 中性偏紧           │
│ 货币政策有效性    │ 低（推绳子）       │ 中（约束通胀）      │
│ 核心矛盾         │ 内需崩塌+成本挤压   │ 通胀粘性+增长放缓   │
│ 资产含义         │ 债券牛/权益承压     │ 短端高利率/长端波动 │
└──────────────────┴──────────────────┴──────────────────┘
```

### 资本流动含义

中国需要弱人民币来支撑出口和对抗通缩，但PPI上行和外部压力制约汇率空间；美国需要强利率来压制通胀。这种政策路径背离驱动全球资本从中国向美国回流。

## 四、核心风险矩阵

| 风险情景 | 概率评估 | 触发条件 |
|----------|---------|---------|
| **中国：滞胀恶化** | 中等 | PPI继续向CPI传导，消费进一步萎缩 |
| **中国：通缩陷阱** | 中高 | 出口回落→PPI再转负→通缩螺旋 |
| **美国：不着陆** | 中等 | 通胀反弹→美联储被迫加息→风险资产重估 |
| **美国：硬着陆** | 低-中 | 限制性利率持续过久→信用事件触发衰退 |
| **中美：政策共振恶化** | 低 | 贸易冲突升级或地缘事件 |

## 五、投资含义

1. **中国债券：** 10Y收益率1.74%已定价通缩，但信用崩塌意味着利率仍有下行空间——不过空间收窄，追低需谨慎。
2. **中国权益：** 内需驱动板块面临双杀（需求弱+成本升），出口/制造业板块受益于全球需求但面临PPI成本挤压。整体谨慎。
3. **美国利率：** 通胀粘性意味着短端利率维持高位，长端受财政和期限溢价影响波动。
4. **美元/人民币：** 结构性力量支持美元偏强，人民币有贬值压力但受政策管控。
5. **大宗商品：** 中国能源CGPI +17.7%反映出上游供给约束，但中国内需疲弱限制涨势——分化将持续。

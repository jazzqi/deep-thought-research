# CIO综合研判：中国宏观场景 — 内需通缩型停滞

## 场景分类：LOW GROWTH + DIVERGENT INFLATION + ACCOMMODATIVE BUT INEFFECTIVE POLICY
### 子类：内需通缩型停滞（Domestic Demand Deflationary Stagnation）
### 置信度：80%
### 时间跨度：当前至2026年末
### 共识水平：HIGH

---

## 一、增长侧（Growth）：LOW — 置信度 85%

| 指标 | 最新值（2026年6月） | 信号 |
|------|--------|------|
| GDP同比（2026 H1） | **4.7%** | 低于5%目标，动能持续走弱 |
| 制造业PMI | **50.3** | 荣枯线边缘，微弱扩张 |
| 非制造业PMI | **50.2** | 同样仅微弱扩张 |
| 零售总额同比 | **1.0%** | 消费几乎停滞 |
| 固定资产投资同比 | **-15.6%** | ⚠️ 断崖式收缩，历史低位 |
| 工业产出 | **5.3%** | 相对稳健，出口驱动 |
| 出口同比 | **27.0%** | 强劲但可持续性存疑 |
| 进口同比 | **36.0%** | 大宗商品拉动的虚高 |
| 消费者信心（5月） | **89.9** | 持续悲观（<100） |
| 新增贷款同比 | **-25.2%** | ⚠️ 信用需求崩塌 |

**核心判断**：经济表面增长4.7%掩盖了严重的结构性问题。固定资产投资-15.6%是除疫情初期外罕见的收缩幅度，反映房地产和基建双弱的深度困境。零售仅1.0%叠加消费者信心89.9，说明居民资产负债表受损严重——房地产财富效应消退是核心拖累。唯一亮点是出口（27%）和进口（36%），但出口的高波动性和贸易摩擦风险不可忽视。工业产出5.3%主要靠出口拉动，内需贡献微乎其微。

---

## 二、通胀侧（Inflation）：DIVERGENT — 置信度 80%

| 指标 | 最新值（2026年6月） | 信号 |
|------|--------|------|
| CPI同比 | **1.0%** | 温和，远低于3%目标 |
| PPI同比 | **4.1%** | ⚠️ 从通缩大幅转正 |
| 能源CGPI同比 | **17.7%** | 能源成本飙升 |
| 农产品CGPI同比 | **-4.9%** | 农产品通缩 |

**核心判断**：这是当前中国宏观最复杂、最值得警惕的部分。呈现罕见的**"上游涨价、下游通缩"格局**：

- **消费者端（CPI 1.0%）**：温和偏低，核心通缩压力持续——房地产财富效应消退、就业市场疲软压制需求端通胀
- **生产者端（PPI 4.1%）**：从之前持续的深度通缩（-3%~-4%）突然大幅转正，能源CGPI同比17.7%是核心驱动
- **传导机制断裂**：上游成本上升无法向下游转嫁，中下游企业利润率被双向挤压（成本升+需求弱），本质是一种"隐性税"——这在全球主要经济体中极为罕见

## 三、政策侧（Policy）：ACCOMMODATIVE BUT INEFFECTIVE — 置信度 90%

| 指标 | 最新值 | 信号 |
|------|--------|------|
| 1年期LPR | **3.0%** | 历史低位 |
| SHIBOR 3M | **1.43%** | 流动性极度充裕 |
| 10Y国债收益率 | **1.7404%** | 极低，反映增长悲观预期 |
| 2Y国债收益率 | **1.2645%** | 短端极低 |
| 10Y-2Y利差 | **0.4759%** | 正利差，曲线正常化 |
| M2同比（6月） | **8.0%** | 货币供给尚可 |
| 美元/人民币 | **6.7775** | 汇率相对稳定 |

**核心判断**：货币政策极度宽松（LPR 3.0%、SHIBOR 1.43%、10Y 1.74%均处于历史低位），但呈现典型的**流动性陷阱**特征：

1. **宽货币无法转化为宽信用**：M2增长8%但新增贷款同比-25.2%，货币在金融体系空转
2. **私人部门去杠杆**：房地产持续下行导致企业和居民都不愿加杠杆
3. **财政空间受限**：土地出让收入大幅下滑严重制约地方财政能力
4. **10Y国债1.74%**：市场已定价深度悲观的增长预期，隐含对未来1-2年零通胀甚至通缩的定价

## 四、CIO综合判断：三组深层背离构成核心矛盾

### 1️⃣ 外热内冷（External vs Domestic Divergence）
```
出口 27%  vs  零售 1.0%
进口 36%  vs  FAI -15.6%
```
经济呈现"两张皮"：外需驱动的工业产出（5.3%）与内需驱动的消费/投资形成巨大鸿沟。一旦全球贸易环境恶化，出口这条"生命线"断裂，整体增长将面临严峻考验。

### 2️⃣ 上游通胀 vs 下游通缩（Upstream Inflation vs Downstream Deflation）
```
PPI 4.1%（上游涨价）  vs  CPI 1.0%（下游通缩）
能源CGPI 17.7%       vs  农产品CGPI -4.9%
```
这是最危险的信号。中下游制造业企业（尤其是中小民营企业）正遭受"成本上升+需求萎缩"的双重挤压。利润率压缩将导致更多企业减产、裁员，进一步压制消费和投资，形成负向螺旋。

### 3️⃣ 宽货币 vs 紧信用（Liquidity Abundance vs Credit Crunch）
```
M2 8.0%（货币充裕）  vs  新增贷款 -25.2%（信用崩塌）
LPR 3.0%（历史低位） vs  消费者信心 89.9（极度悲观）
```
流动性陷阱的特征非常经典——"你可以把马牵到水边，但不能强迫它喝水"。私人部门主动去杠杆的行为，使货币政策传导机制彻底失效。

## 五、投资含义

### 对债券的启示
- 10Y国债1.74%虽然已处于极低位，但在通缩压力持续的背景下，利率仍有下行空间
- 收益率曲线正常化（10Y-2Y利差47bp）反映市场对长期增长的悲观，而非对通胀的担忧
- **偏好：长久期利率债**

### 对权益的启示
- 上游能源相关板块有PPI支撑，但须警惕大宗商品价格回落风险
- 下游消费、地产相关板块面临双重挤压，盈利预期下调风险大
- 出口导向型制造业短期景气但波动性高，贸易摩擦是尾部风险
- **偏好：防御性配置，规避内需敏感板块**

### 对汇率的启示
- 人民币6.7775相对稳定，但在经济基本面弱势下，中期贬值压力存在
- 若出口走弱，央行可能允许人民币适度贬值以支持出口竞争力

### 对政策的期待
当前场景下，常规货币政策已失效。需要的是：
1. **财政扩张**——中央政府直接加杠杆，通过转移支付提振居民收入
2. **结构性改革**——打破房地产-土地财政的死循环
3. **非常规宽松**——可能的量化宽松或直接向居民部门转移支付

---

## 六、风险提示

- **上行风险**：若出台超预期的大规模财政刺激（如消费券、直接补贴），可能打破通缩螺旋
- **下行风险**：出口断崖式下滑（全球衰退或贸易摩擦升级）将导致增长跌破4%
- **尾部风险**：房地产市场硬着陆引发系统性金融风险

---

## 结论

当前中国宏观经济处于**内需通缩型停滞**，这是最棘手的宏观场景之一。传统货币政策工具已基本失效（流动性陷阱），需要财政政策和结构性改革的强力介入。市场已经通过10Y国债1.74%定价了深度悲观预期，但尚未完全定价"政策失灵"的风险。投资者应保持防御性配置，关注政策拐点信号，同时警惕出口端的尾部风险。

**共识水平：HIGH** — 分析师在增长评估（LOW）、通胀特征（DIVERGENT）、政策效果（INEFFECTIVE）三个维度上观点高度一致，分歧仅在于对政策出台时点和力度的预期差异。

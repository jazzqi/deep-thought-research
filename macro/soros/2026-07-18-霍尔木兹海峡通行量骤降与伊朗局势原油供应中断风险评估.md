# 霍尔木兹海峡通行量骤降与伊朗局势：原油供应中断风险评估

## 概述

一个新的宏观叙事正在浮现：**霍尔木兹海峡船舶通行量骤降，伊朗局势紧张加剧原油供应中断风险**（信号强度：0.50）。本报告从多维度评估这一叙事的可信度、传导路径及潜在影响。

---

## 1. 霍尔木兹海峡的战略意义

霍尔木兹海峡是全球最重要的石油运输咽喉要道：
- 承载全球约 **20-30%** 的原油及石油制品运输（约1700-2100万桶/日）
- 连接波斯湾产油国（沙特、伊朗、伊拉克、阿联酋、科威特等）与全球市场
- 最窄处仅约33公里宽，极易被封锁或干扰
- 无有效替代路线：绕行好望角将增加数千公里航程和数周时间

## 2. 数据支撑与交叉验证

### 2.1 中国能源价格指数(CGPI) — 强烈信号

| 指标 | 数值 | 时间 |
|------|------|------|
| CGPI能源同比 | **+17.74%** | 2026年6月 |
| CGPI农产品同比 | -4.94% | 2026年6月 |
| PPI同比 | **+4.1%** | 2026年6月 |
| CPI同比 | +1.0% | 2026年6月 |

中国CGPI（企业商品价格指数）中能源分类同比飙升 **+17.74%**，这是近年来的高位。能源价格的强劲上涨与供应端担忧高度一致。同期农产品价格（-4.94%）则在下行，说明这不是全面的通胀，而是 **能源板块特有的供给驱动型价格上涨**。

### 2.2 宏观背景

| 指标 | 数值 | 时间 |
|------|------|------|
| 美国CPI同比 | 3.5% | 2026年6月 |
| 美国核心CPI | 0.3%（月环比） | 2026年6月 |
| 美国10Y国债收益率 | 4.57% | 2026年7月 |
| 美元/人民币 | 6.78 | 2026年7月 |

美国通胀仍处于3.5%的较高水平，远未达到美联储2%目标。**如果霍尔木兹危机推升原油价格，将对仍在抵抗通胀的全球经济构成二次冲击。**

### 2.3 叙事缺失的发现

数据库搜索未发现任何关于霍尔木兹海峡/伊朗原油供应中断的既有叙事记录。这意味着：
- 这是一个 **新兴的、未被市场充分定价** 的叙事
- 团队拥有先发优势来研究这一主题
- 可能需要从外部数据源（如Vortexa、Kpler等油轮跟踪数据）获取更多验证

## 3. 影响传导路径

### 3.1 直接路径
```
霍尔木兹通行量骤降 → 波斯湾原油出口受阻 → 全球原油供应缺口扩大
→ 原油现货/期货价格飙升 → 能源板块上涨 → 下游行业成本上升
```

### 3.2 二级传导
```
油价飙升 → 汽油/柴油/航油价格上涨 → 运输成本上升 → 消费品通胀
→ 各国央行面临更严峻的抗通胀挑战 → 货币政策被迫维持紧缩
→ 经济衰退风险上升
```

### 3.3 中国特定影响
```
中国是全球最大原油进口国（约1100万桶/日）→ 进口成本急剧上升
→ PPI上行压力 → 制造业利润压缩 → 出口竞争力下降
→ 人民币贬值压力 → 能源安全焦虑上升
```

霍尔木兹海峡通行量骤降与伊朗局势紧张构成的原油供应中断叙事，**信号强度0.50属于中等置信度，但潜在影响极大**，属于"高影响、中等概率"的尾部风险事件。值得团队投入资源进行深度跟踪。

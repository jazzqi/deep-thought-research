## 三、场景演绎与核心矛盾

### 核心矛盾：PPI加速 vs 增长失速

中国目前面临一个经典的政策两难：

```
            ┌─────────────────────────────────────┐
            │         政策两难三角                  │
            │                                      │
            │   增长疲弱 (GDP 4.7%)                │
            │       ↓ 需要更多刺激                  │
            │   宽松政策 (LPR 3.0%)                │
            │       ↓ 可能推升PPI                  │
            │   PPI加速 (4.1%) ← 能源成本+         │
            │       ↓ 挤压企业利润                  │
            │   投资收缩 (FAI -15.6%)              │
            │       ↓ 进一步拖累增长                │
            └─────────────────────────────────────┘
```

**三个可能的演化路径**:

1. **基准路径 (60%)**: PPI高位维持但CPI保持温和 → 央行继续温和宽松，财政加大发力 → 增长低位企稳但不会V型反弹
2. **滞胀路径 (25%)**: 能源价格持续高企 -> PPI向CPI传导 -> CPI突破2.5% -> 央行被迫边际收紧 -> 增长进一步承压
3. **通缩路径 (15%)**: 能源价格回落 + 需求持续疲弱 -> PPI回落 -> 全面通缩压力 -> 需超大规模刺激

### 债券市场隐含的定价

10Y国债1.74%是全球主要经济体中的极低水平。这反映了：
- 市场定价经济增长将持续低于潜在增速
- 通胀预期极低（CPI仅1.0%）
- 市场预期央行将进一步降息

但如果PPI持续加速并向CPI传导，1.74%的收益率将面临重定价风险。

## 四、投资含义

1. **债券**: 短期因增长疲弱仍有支撑，但中期PPI向CPI传导风险是主要威胁。久期策略需谨慎。
2. **权益**: 增长放缓+PPI挤压利润=盈利下行周期。关注受益于PPI上行的上游能源板块，回避中游制造。
3. **汇率**: 增长减速+利差收窄，人民币面临温和贬值压力。
4. **商品**: 能源CGPI +17.7%说明能源品仍有上行风险。但工业金属受FAI -15.6%拖累，需求端逻辑偏弱。
5. **房地产**: FAI -15.6% + 新贷款 -25.2% + 房价持平 = 行业仍在寻底。

# 当前宏观场景分析报告

**分析日期**: 2026年7月19日  
**分析师**: Soros  

---

## 一、核心结论

**场景分类**: **中国处于「低增长 + 低通胀 + 极度宽松政策」区间，逼近流动性陷阱/资产负债表衰退边缘**  
**置信度**: 85%  
**时间跨度**: 未来6-12个月  

---

## 二、数据支撑

### 2.1 经济增长（Growth）—— 全面走弱，低于目标

| 指标 | 最新值 | 趋势判断 |
|------|--------|----------|
| **GDP H1 2026** | **+4.7% YoY** | 低于全年目标5.0%，较2025全年5.0%和H1 2025的5.3%持续放缓 |
| **制造业PMI (6月)** | **50.3** | 仅略高于荣枯线，边际扩张极弱 |
| **非制造业PMI (6月)** | **50.2** | 同样紧贴荣枯线，服务业动能不足 |
| **零售销售 (6月)** | **+1.0% YoY** | 消费近乎停滞，远低于疫情前水平 |
| **FAI (6月)** | **-15.6% YoY** | 固定资产投资急剧萎缩，房地产和基建双重拖累 |
| **工业增加值** | 5.7%（旧数据） | 制造业生产端尚可，但需求端信号恶化 |
| **消费者信心 (5月)** | **89.9** | 低于100荣枯线，信心低迷抑制消费意愿 |

**关键判断**: 增长动能在2026年上半年显著失速。固定资产投资-15.6%是严重的衰退信号，零售仅+1.0%表明消费无法对冲投资下滑。GDP从5.3%→5.0%→4.7%的轨迹清晰下行。

### 2.2 通货膨胀（Inflation）—— 温和但结构分化

| 指标 | 最新值 | 趋势判断 |
|------|--------|----------|
| **CPI (6月)** | **+1.0% YoY** | 温和偏低，无通胀压力，但有通缩隐忧 |
| **CPI (5月)** | +1.2% YoY | 环比略有放缓 |
| **PPI (6月)** | **+4.1% YoY** | 由负转正，主要受能源成本推动 |
| **CGPI农业** | **-4.94% YoY** | 农产品价格深度通缩 |
| **CGPI能源** | **+17.74% YoY** | 能源价格大幅上涨，输入性成本压力 |

**关键判断**: **通胀不是当前故事——通缩才是核心风险。** CPI仅1.0%且环比-0.3%，低于2%的理想水平。PPI转正至4.1%是结构性的能源成本推动（国际油价、煤炭等），而非需求拉动。农产品大幅通缩说明农村消费力极弱。整体环境是**温和通缩型**而非通胀型。

### 2.3 货币与信贷（Monetary & Credit）—— 极度宽松但传导失灵

| 指标 | 最新值 | 趋势判断 |
|------|--------|----------|
| **10Y国债收益率** | **1.74%** | 历史低位，反映极度悲观的经济预期 |
| **2Y国债收益率** | **1.26%** | 短端利率极低，货币宽松程度罕见 |
| **10Y-2Y利差** | **+47.6bp** | 收益率曲线陡峭，市场定价宽松预期的延续 |
| **SHIBOR 3M** | **1.43%** | 银行间流动性极其充裕 |
| **1年期LPR** | **3.00%** | 贷款利率持续处于低位 |
| **新增贷款 (6月)** | **-25.2% YoY** | 信用需求崩塌式收缩 |
| **社融** | 6245亿（4月） | 信贷供给充裕但需求不足 |

**关键判断**: **典型的「宽货币+紧信用」格局。** 利率处于历史最低位，流动性极度充裕（SHIBOR 1.43%），但信贷需求却创下-25%的同比萎缩。这不是货币政策不够松的问题，而是经济体陷入 **资产负债表衰退** 的特征——企业/居民去杠杆、不借钱。收益率曲线陡峭是市场在定价"利率将长期低位"。

## 三、场景分类矩阵

| 维度 | 状态 | 信号强度 |
|------|------|----------|
| **Growth** | 弱（低于潜在增长率） | ⚠️⚠️⚠️ |
| **Inflation** | 低（通缩风险 > 通胀风险） | ⚠️⚠️ |
| **Policy** | 极度宽松（价格工具/数量工具均已到位但传导不畅） | ⚠️⚠️⚠️ |
| **Credit** | 严重收缩（资产负债表衰退特征） | ⚠️⚠️⚠️⚠️ |
| **External** | 出口尚可但贸易摩擦持续 | ⚠️⚠️ |

**场景确定**: **Growth=Down, Inflation=Down, Policy=Accommodative**  
**子类**: 流动性陷阱型资产负债表衰退（类似日本1990s-2000s初期）

## 五、风险因素与不确定性

1. **PPI向CPI传导风险**: 能源CGPI+17.74%，若持续可能推升成本型通胀，改变低通胀格局（概率20%）
2. **政策刺激超预期**: 若推出大规模财政刺激（特别国债/消费券），可能改变增长轨迹（概率30%）
3. **外部冲击**: 科技封锁/贸易摩擦升级可能进一步打击出口和制造业（概率25%）
4. **房地产市场企稳**: 若房价企稳且信心恢复，信用收缩可能缓解（概率20%）

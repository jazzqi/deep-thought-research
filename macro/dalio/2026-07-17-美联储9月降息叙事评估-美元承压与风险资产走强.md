## 三、传导机制分析

### 3.1 降息→美元走弱传导链

```
美联储鸽派信号 → 利率预期下降 → 美元资产吸引力降低 → 美元走弱 → 新兴市场/风险资产受益
```

**置信度**: 中等偏高。历史数据表明降息预期与美元走势存在高度相关性。当前USD/CNY已经体现出这一趋势。

### 3.2 降息→风险资产传导链

```
降息预期 → 无风险利率下降 → 风险溢价收缩 → 权益类资产估值提升 → 风险资产走强
```

**置信度**: 中等。需要关注：
- 如果降息是"预防式"而非"衰退式"，对风险资产利好更显著
- 当前PMI在50以上，经济未进入衰退，符合"预防式降息"场景

### 3.3 对中国资产的影响

美元走弱→人民币企稳→：
1. **A股**: 外资回流预期增强，北向资金可能增加
2. **港股**: 更直接受益于美元流动性宽松
3. **人民币汇率**: 贬值压力减轻，有利于稳定资本流动
4. **大宗商品**: 美元计价商品（黄金、铜等）受益

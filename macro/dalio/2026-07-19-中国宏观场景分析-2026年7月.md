# 中国宏观场景分析 — 2026年7月

## 核心结论

**当前场景分类：弱增长 + 温和CPI通胀 + PPI快速再通胀 + 宽松但失效的政策**

**置信度：75%**

---

## 一、增长（Growth）：明确走弱

| 指标 | 最新值 | 数据期 | 趋势 |
|------|--------|--------|------|
| GDP同比 | **4.7%** | 2026 H1 (Q1-Q2) | 从2025年5.0%放缓，Q1为5.0%，H1降至4.7% |
| 制造业PMI | **50.3** | 2026年6月 | 勉强扩张，5月曾降至50.0 |
| 非制造业PMI | **50.2** | 2026年6月 | 同样处于荣枯线边缘 |
| 零售额同比 | **1.0%** | 2026年6月 | 极弱，消费几乎零增长 |
| 固定资产投资同比 | **-15.6%** | 2026年6月 | **断崖式收缩**，为多年罕见的深度负值 |
| 消费者信心指数 | **89.9** | 2026年5月 | 低于100，持续处于悲观区间 |

**增长判断：** GDP增速已从2025年全年5.0%放缓至2026上半年4.7%，且二季度较一季度进一步走弱。PMI指数在50.0-50.3区间徘徊，实质处于停滞边缘。固定资产投资-15.6%的同比跌幅是极其令人担忧的信号——投资端正在经历深度收缩。零售仅增长1.0%，消费动力严重不足。**增长处于"类衰退"（growth recession）状态。**

---

## 二、通胀（Inflation）：CPI温和 vs PPI飙升——结构性分歧

### CPI：温和偏弱
| 月份 | CPI同比 |
|------|---------|
| 2026年2月 | 1.3% |
| 2026年3月 | 1.0% |
| 2026年4月 | 1.2% |
| 2026年5月 | 1.2% |
| 2026年6月 | **1.0%** |

CPI持续运行在1.0-1.3%区间，显著低于中国央行3%的政策目标。环比来看6月为-0.3%，边际走弱。

### PPI：快速攀升
| 月份 | PPI同比 |
|------|---------|
| 2026年2月 | **-0.9%** |
| 2026年3月 | 0.5% |
| 2026年4月 | 2.8% |
| 2026年5月 | 3.9% |
| 2026年6月 | **4.1%** |

PPI在短短4个月内从通缩(-0.9%)飙升至通胀4.1%，这是**非常剧烈的再通胀**。驱动因素中，能源类CGPI同比+17.74%是主要推手，而农产品CGPI同比-4.94%形成拖累。

### 通胀结构判断
- **CPI端：** 通缩压力仍在，1.0%的读数意味着内需严重不足
- **PPI端：** 上游成本冲击明显，能源价格上涨正沿着工业链条传导
- **核心风险：** PPI向CPI的传导是否会发生？如果PPI持续高企而终端需求无法恢复，企业利润将受到严重挤压

## 三、政策（Policy）：宽松但失效

| 指标 | 最新值 | 解读 |
|------|--------|------|
| 1年期LPR | **3.0%** | 历史低点附近 |
| SHIBOR 3M | **1.43%** | 极低，流动性充裕 |
| 10Y国债收益率 | **1.7404%** | 历史低位，反映极度悲观的增长预期 |
| 2Y国债收益率 | **1.2645%** | 极度平坦 |
| 10Y-2Y利差 | **47.6bp** | 正但不算陡峭 |
| 新贷款同比 | **-25.2%** | 信用创造剧烈收缩 |
| 社融 | 6245.0亿 | 2026年4月值 |

### 政策判断

货币政策名义上是宽松的——利率处于低位，流动性充裕。但问题在于**信用需求端崩塌**：
- 新贷款同比下降25.2%，说明即使利率很低，企业和居民也不愿意借钱
- FAI -15.6%说明投资意愿极弱
- 这是典型的"推绳子"（pushing on a string）场景——央行可以提供流动性，但无法强迫经济主体借贷和投资

**债券市场定价**是对增长最真实的投票：10Y国债收益率1.74%已隐含了长期低增长、低通胀的预期。

## 四、场景综合分类

### 矩阵定位

```
               通胀（CPI）
             低 ←━━━→ 高
        ┌─────────────────┐
  强    │                   │ 黄金期
        │                   │
增      │  当前场景          │
长      │  (弱增长,CPI温和)  │
        │                   │
  弱    │                   │ 滞胀
        └─────────────────┘
```

但考虑到PPI端的情况，更准确的描述是：

**"CPI通缩型弱增长 + PPI工业再通胀 + 政策传导断裂"**

这是一个非经典的混合 regime：
1. **从CPI角度看：** 通缩性衰退（Disinflationary Recession）
2. **从PPI角度看：** 上游成本推动型通胀正在积累（Pipeline Inflation Building）
3. **从政策角度看：** 流动性陷阱特征（Liquidity Trap）——利率工具失效

### 最接近的经典场景
**"通缩性放缓（Disinflationary Slowdown）主导，叠加工业再通胀（Industrial Reflation）风险"**

## 五、关键风险与前瞻

### 上行风险（Bull Case）
- 政策刺激加码（财政发力）带动信用企稳
- 出口韧性维持增长底线
- PPI上涨最终传导至企业利润改善和补库存

### 下行风险（Bear Case）
- FAI继续塌缩引发就业-收入-消费的负向螺旋
- PPI高涨而CPI低迷 → 企业利润挤压 → 更多企业违约
- 房地产和地方债务问题进一步恶化信用环境
- 通缩预期自我实现

### 需关注的信号
1. **信用底：** 新贷款和社融何时触底回升？（领先指标）
2. **PPI-CPI剪刀差：** 能否收窄？收窄方向是PPI回落还是CPI回升？
3. **财政政策：** 是否有大规模财政刺激出台？
4. **房地产市场：** 房价指数0.5%接近持平，销售面积能否回暖？

## 六、对投资组合的启示

1. **债券：** 1.74%的10Y收益率已计入大量悲观预期，进一步下行空间有限但短期未见逆转动力
2. **股票：** 弱增长+信用收缩的环境不利于风险资产，关注结构性机会（受益于PPI上行的上游行业）
3. **商品：** PPI再通胀+能源CGPI +17.7%支撑工业品价格，但需求端疲弱形成制约
4. **汇率：** 弱增长+低利率环境对人民币形成贬值压力

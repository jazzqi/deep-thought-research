# 霍尔木兹海峡通行量骤降与原油供应中断风险评估

## 概述

一个新的宏观叙事正在浮现：**霍尔木兹海峡船舶通行量骤降，伊朗局势紧张加剧原油供应中断风险**（叙事强度 0.50）。本报告从专业分析师视角评估该叙事的可信度、传导路径及投资含义。

---

## 1. 霍尔木兹海峡的战略地位

霍尔木兹海峡是全球最重要的石油运输咽喉要道：
- **全球约20-25%的石油运输**（约1700-2000万桶/日）经过该海峡
- 连接波斯湾产油国（沙特、伊朗、伊拉克、阿联酋、科威特、卡塔尔）与全球市场
- 宽度仅约33公里（最窄处），极易被封锁或袭扰
- 历史上1980年代"油轮战争"期间曾受严重威胁

若通行量确实骤降，将直接影响全球原油供应的20%以上，其严重程度远超利比亚或尼日利亚等单一国家的供应中断。

2. **通胀环境脆弱**：美国CPI同比3.5%，核心CPI环比0.3%，美联储距离2%目标仍有距离。原油供应冲击将直接推高汽油价格和运输成本，可能导致通胀二次抬升，迫使美联储维持更长时间的高利率。

## 3. 叙事可信度评估

### 支持该叙事的论据

1. **伊朗局势持续紧张**：伊朗核问题谈判陷入僵局，以色列-伊朗代理人冲突不断升级，美国对伊制裁持续收紧
2. **先例可循**：2019年阿曼湾油轮遇袭、2020年苏莱曼尼事件后伊朗曾短暂威胁海峡通行
3. **保险市场信号**：若战争风险保险费率飙升，船东可能主动减少通行
4. **替代路线有限**：沙特和阿联酋虽有东西向输油管道，但总运力远不及海峡通行量

### 削弱该叙事的论据

1. **伊朗全面封锁海峡可能性较低**：完全封锁等同于对美开战，不符伊朗战略利益
2. **中国和俄罗斯的外交调解**：中俄均有动机防止海峡危机失控
3. **战略石油储备可作为缓冲**：IEA成员国和中国持有大量战略储备
4. **叙事强度仅0.50**：属于中等信号，尚未形成确定性趋势

### 综合判断

**置信度：0.50（中等）**。这一叙事目前处于"值得关注但尚未确认"的阶段。关键在于：
- 通行量骤降的具体幅度和持续时间
- 降量是主动规避（保险公司提高保费）还是被动受阻（军事行动）
- 伊朗与美国的外交/军事互动

### 对中国的影响

中国是全球最大原油进口国，2025年日进口量约1100-1200万桶，其中约40-50%经过霍尔木兹海峡。影响路径：
1. **输入性通胀**：石油价格上涨直接推升PPI，进而传导至CPI
2. **能源安全**：需加速释放战略储备，增加俄罗斯、非洲、美洲采购
3. **人民币汇率压力**：进口成本上升=贸易条件恶化
4. **一带一路影响**：中国-伊朗能源合作项目面临不确定性

## 5. 需要进一步跟踪的关键信号

1. **船舶自动识别系统（AIS）数据**：霍尔木兹海峡实际通行船舶数量变化
2. **原油期货期限结构**：近月合约相对于远月合约的升水幅度（backwardation加深 = 供应紧张）
3. **油轮战争风险保险费率**：该费率是衡量实际航行风险的灵敏指标
4. **伊朗外交动态**：伊核协议谈判、联合国大会期间的外交互动
5. **美国战略石油储备（SPR）变动**：若SPR开始释放，验证供应紧张判断
6. **中国原油进口数据**：月度进口量及来源国变化

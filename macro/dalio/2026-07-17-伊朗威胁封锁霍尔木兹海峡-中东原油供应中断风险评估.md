# 伊朗威胁封锁霍尔木兹海峡：中东原油供应中断风险评估

2026年7月17日，伊朗军方发出严厉警告，声称"只要美国继续，一滴油都不会流出中东"。这一声明标志着一项重大地缘政治风险的升级——霍尔木兹海峡封锁威胁。本报告从专业分析师视角评估该事件的影响和后续推演。

## 影响分析

### 1. 原油价格冲击
- **基准情景（概率40%）**：口头威胁持续，油价短期冲高5-10美元后回落
- **升级情景（概率40%）**：伊朗实施有限封锁/袭船，油价突破100美元
- **全面封锁情景（概率20%）**：海峡实质性封锁，油价可能飙升至120-150美元

### 2. 对中国的影响（最直接）
- 中国是全球最大原油进口国，中东约占中国进口量的45-50%
- 现有战略石油储备（SPR）约覆盖90天进口量
- PPI可能从当前-3.6%迅速转正，传导至CPI
- 制造业成本上升，PMI可能跌破50荣枯线
- 人民币汇率（当前6.78）承压

### 3. 对美国的影响
- 美国已是净石油出口国，但油价上涨仍会推高汽油价格
- 通胀预期上升 → 10Y收益率上行 → 风险资产承压
- 消费者信心（当前97.4）将受打击

### 4. 全球市场传导
- **受益**：能源股、航运股、军工、黄金
- **受损**：航空、运输、化工、制造业
- 避险情绪升温，美元走强，新兴市场承压

## 结论与建议

### 核心判断
伊朗封锁霍尔木兹海峡的叙事已从理论讨论进入实际行动信号阶段。强度0.50意味着发生概率不低，且市场尚未充分定价。鉴于中国是最大的潜在受害者（中东原油进口占比高），这一叙事值得团队投入持续跟踪资源。

# 全球宏观场景分析：中美大分化与Regime Divergence

**分析日期**: 2026年7月18日  
**分析师**: Dalio  
**置信度**: 0.80  
**时间跨度**: 6-12个月

---

## 一、核心结论

当前全球宏观最核心的特征是**中美两大经济体的regime全面分化（divergence）**：

| 维度 | 🇺🇸 美国 | 🇨🇳 中国 |
|------|---------|---------|
| **增长** | 晚期扩张（GDP 3.3%） | 资产负债表衰退（GDP 4.7%，但内需极弱） |
| **通胀** | 粘性高通胀（CPI 3.5%，核心CPI MoM 0.3%） | 通缩压力（CPI 1.0%，远低于目标） |
| **货币政策** | 限制性（10Y 4.57%，2Y 4.16%） | 宽松但信用坍缩（10Y 1.74%，LPR 3.0%） |
| **信用周期** | 尚可 | 严重收缩（新贷款-25.2%，固投-15.6%） |
| **汇率** | 强势美元 | 人民币承压（USD/CNY ~6.78） |

这组数据指向一种**不可持续的分化状态**，最终会通过某种方式收敛——要么美国经济放缓迫使美联储降息，要么中国经济刺激加码带动内需回暖。理解这种分化的演变路径是判断未来6-12个月资产价格走势的关键。

---

## 二、美国场景分析：Late Cycle Expansion with Sticky Inflation

### 2.1 增长状态

| 指标 | 数值 | 评判 |
|------|------|------|
| GDP同比 | 3.3%（2026年6月） | 高于潜在增速，但可能为周期尾部 |
| 失业率 | 4.2% | 从底部回升，但仍处低位 |
| 非农就业 | +7.3万（6月） | 就业增长放缓但仍为正 |
| 初请失业金 | 23.7万 | 仍处低位，劳动力市场尚未恶化 |
| 工业产出 | -0.1% | ⚠️ 制造业已出现萎缩信号 |
| 零售销售MoM | +0.5% | 消费尚有韧性 |
| 消费者信心 | 97.4 | 相对稳定 |

**增长判断**: 美国经济整体仍在扩张，但已进入**晚期周期**。制造业已开始萎缩（工业产出负增长），但服务业和消费仍有韧性。劳动力市场从极度紧张状态正常化（失业率从3.4%低点升至4.2%），但尚未崩塌。

### 2.2 通胀状态

| 指标 | 数值 | 评判 |
|------|------|------|
| CPI同比 | 3.5% | 仍显著高于2%目标 |
| 核心CPI MoM | 0.3% | 折年率约3.6%，下行停滞 |

**通胀判断**: 通胀回落进程已**明显停滞**。3.5%的CPI同比距离美联储2%目标还有很大距离，而核心CPI环比0.3%意味着折年率仍高于3.5%。**"最后一公里"的去通胀是最艰难的**——因为剩下的通胀主要由住房租金和服务业工资驱动，对利率不敏感。

### 2.3 政策与利率

| 指标 | 数值 |
|------|------|
| 10年期国债收益率 | 4.57% |
| 2年期国债收益率 | 4.16% |
| 利差(2s10s) | +0.41%（正斜率） |

**政策判断**: 
- 收益率曲线在经历了长时间倒挂后已**恢复正常正斜率**（+41bp），这是历史上晚周期扩张的典型特征
- 4.57%的10Y收益率和4.16%的2Y收益率意味着市场定价"higher for longer"
- 曲线正常化并不意味着降息即将到来——相反，历史上曲线转正后往往意味着经济仍有过热风险

### 2.4 美国场景分类

> **场景: Late Cycle Expansion with Sticky Inflation（晚期扩张+粘性通胀）**
> - **置信度: 0.80**
> - **风险情景**: 如果通胀无法继续下行（CPI卡在3%以上），美联储将无法降息，高利率持续时间超预期，最终导致经济硬着陆

## 三、中国场景分析：Balance Sheet Recession with Deflationary Pressure

### 3.1 增长状态

| 指标 | 数值 | 评判 |
|------|------|------|
| GDP同比 | 4.7%（2026 H1） | 低于潜在增速~5%+，且依赖外需 |
| 制造业PMI | 50.3 | 仅略高于荣枯线，扩张乏力 |
| 非制造业PMI | 50.2 | 服务业同样疲弱 |
| 工业产出 | 5.3% | 生产端尚可，但需求端更弱 |
| 零售销售同比 | 1.0% | ⚠️ 消费极度疲弱 |
| 固定资产投资同比 | -15.6% | ⚠️ 投资崩塌，房地产是主因 |
| 消费者信心 | 89.9 | 低迷 |
| 出口同比 | 27.0% | ✅ 唯一亮点，但为外需驱动 |
| 进口同比 | 36.0% | 部分反映能源/商品价格 |

**增长判断**: 中国经济增长呈现**严重的内外分化**。出口强劲（+27%）与外需相关，但国内需求全面塌陷：零售仅1%、固投-15.6%、消费者信心不足90。这不只是周期性疲弱，而是**典型的资产负债表衰退特征**——私营部门在去杠杆，不愿借贷消费和投资。

### 3.2 通胀状态

| 指标 | 数值 | 评判 |
|------|------|------|
| CPI同比 | 1.0% | ⚠️ 远低于3%目标，通缩风险 |
| PPI同比 | 4.1% | 主要受能源价格驱动（CGPI能源+17.7%） |
| CGPI农产品同比 | -4.9% | 食品通缩 |

**通胀判断**: 整体处于**通缩区间**。核心CPI极低（1.0%），远低于央行3%的目标。PPI上涨是能源成本推动而非内需拉动（CGPI能源+17.7% vs 农产品-4.9%）。剔除能源后，工业品价格实际在下跌。这是一种"坏的通胀"——成本推动而非需求拉动。

### 3.3 政策与信用

| 指标 | 数值 | 评判 |
|------|------|------|
| 1年期LPR | 3.0% | 历史低位 |
| SHIBOR 3M | 1.43% | 流动性充裕 |
| 10年期国债收益率 | 1.74% | ⚠️ 极低，反映通缩预期 |
| 2年期国债收益率 | 1.27% | 同样极低 |
| 利差(10Y-2Y) | +0.47% | 正斜率，但处于低利率环境 |
| M2同比 | 8.0% | 货币供给尚可 |
| 新增贷款同比 | -25.2% | ⚠️ 信用严重收缩 |
| 社会融资规模 | 6245亿 | 融资需求不足 |
| USD/CNY | ~6.78 | 人民币有贬值压力 |

**政策判断**: 这组数据揭示了**中国央行面临的经典困境——"推绳子"（pushing on a string）**:
- 货币利率已降至历史低位（LPR 3.0%，SHIBOR 1.43%），但新增贷款暴跌25%
- 债券市场定价通缩（10Y国债仅1.74%，几乎全球最低之一）
- 中美10Y利差约-283bp，人民币承压
- 问题不在于货币不够宽松，而在于**信用需求坍塌**——企业和家庭不愿意借钱

### 3.4 中国场景分类

> **场景: Balance Sheet Recession with Deflationary Pressure（资产负债表衰退+通缩压力）**
> - **置信度: 0.85**
> - **风险情景**: 如果通缩预期自我强化，可能陷入日本化长期停滞。出口强劲是唯一的"救命稻草"，但地缘政治风险可能随时打断外需

## 四、中美利差与汇率核心矛盾

中美10年期国债利差目前约 **-283bp**（中国1.74% vs 美国4.57%），这是历史上罕见的深度倒挂。这意味着：

1. **人民币贬值压力持续存在**——利差如此之大，资本外流动机强烈
2. **中国货币政策空间受限**——如果进一步降息，利差会继续扩大，加大汇率压力
3. **"不可能三角"困境**——中国需要在"独立的货币政策、资本流动、汇率稳定"之间艰难平衡

USD/CNY在6.78附近，考虑到利差水平，人民币实际上面临显著的高估压力。如果中国需要更多宽松来应对内需疲弱，而美国维持高利率，人民币可能会进一步走弱。

## 五、全球宏观合成与资产含义

### 5.1 场景合成的核心矛盾

把中美放在一起看，全球宏观的**根本矛盾**在于：

1. **美国需要经济放缓来降通胀**，但经济仍然有韧性
2. **中国需要经济刺激来抗通缩**，但政策空间受限
3. **高美国利率吸走全球流动性**，加重了中国的资本外流压力
4. **中国输出通缩（廉价出口）**，帮助美国压通胀——这是两国之间微妙的"共生"关系

### 5.2 资产价格的宏观含义

| 资产类别 | 宏观含义 | 方向判断 |
|---------|---------|---------|
| **美债** | 通胀粘性限制下行空间，收益率将维持高位震荡 | 中性偏空 |
| **中债** | 通缩环境下收益率仍有下行空间，但已定价较多 | 看多但空间有限 |
| **美元** | 利差优势+避险需求支撑，但降息预期会压制 | 高位震荡偏强 |
| **人民币** | 利差+通缩双杀，贬值压力持续 | 偏弱，关注7.0关口 |
| **美股** | 估值偏高，利率高企，盈利面临下修风险 | 谨慎 |
| **A股** | 通缩压制估值，但政策底已现，结构性机会 | 结构性分化 |
| **商品** | 中国需求疲弱vs美国服务业韧性，工业金属承压 | 偏弱 |
| **黄金** | 全球货币宽松趋势+去美元化+通缩担忧 | 看多 |

### 5.3 两个key scenario paths

**Path 1 - 美国降通胀+中国刺激（概率约40%）**:
- 美国经济逐步降温，CPI回落至2.5-3%，美联储开始降息
- 中国推出大规模财政刺激（消费补贴+房地产收储）
- 中美利差收窄，人民币企稳
- 全球risk-on，新兴市场受益

**Path 2 - 美国通胀粘性+中国通缩持续（概率约45%）**:
- 美国CPI卡在3%以上，美联储全年不降息
- 中国刺激力度不足，通缩预期自我强化
- 利差持续倒挂，人民币渐进贬值
- 全球risk-off，资金流向美元资产

**Path 3 - 尾部风险：美国硬着陆（概率约15%）**:
- 高利率最终击穿经济，失业率快速上升
- 美联储被迫大幅降息
- 美元走弱，人民币压力缓解
- 中国外需崩塌，但政策刺激加码

## 六、关键监测信号

1. **美国核心CPI MoM**：若持续高于0.25%，则降息无望
2. **中国新增贷款数据**：若持续负增长，通缩螺旋将加深
3. **中国政治局会议/财政信号**：是否有超预期的消费刺激或房地产收储
4. **USD/CNY 7.0关口**：若突破，可能引发资本外流加速
5. **中美利差变化**：任何方向的显著变动都将带来资产价格重定价
6. **美国失业金申请人数**：若跳升至30万+，硬着陆风险上升

## 七、总结

当前全球处于**rare regime divergence**——美国在晚期扩张中与粘性通胀搏斗，中国在资产负债表衰退中与通缩抗争。这种分化不会永远持续。从Dalio的长期债务周期视角看，中国正处于**债务周期下行/去杠杆阶段**，而美国处于**债务周期顶部/紧货币阶段**。两大经济体的不同相位制造了独特的全球宏观环境——没有一个单一的"regime"可以描述全球。

在这种环境下，最危险的投资策略是"押注单一方向"。跨资产、跨市场的分散配置，以及对中美政策转向的敏感度，将是未来6-12个月获取超额收益的关键。

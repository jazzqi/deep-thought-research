# 中国宏观场景分析报告 — 2026年H1

## 一、场景分类结论

| 维度 | 判定 | 置信度 |
|:---|:---|:---:|
| **增长 (Growth)** | **停滞边缘 / 轻度衰退** (Stagflationary倾向偏通缩) | 85% |
| **通胀 (Inflation)** | **通缩** (Deflation) — CPI 0.0%, PPI -3.6% | 90% |
| **政策 (Policy)** | **宽松但失效** (Accommodative but impaired transmission) | 80% |
| **综合场景** | **通缩性停滞 (Disinflationary Stagnation / Balance Sheet Recession)** | 85% |

**核心判断：中国当前处于典型的「资产负债表衰退」区间。货币政策极度宽松但信用创造机制断裂，增长依赖出口唯一引擎，内需呈现全面收缩特征。**

---

## 二、增长维度分析

### 2.1 GDP — 趋势性放缓

| 时期 | GDP同比 |
|:---|:---:|
| 2025全年 | 5.0% |
| 2026Q1 | 5.0% |
| 2026H1 | **4.7%** |

- H1 GDP 4.7% 低于全年"5%左右"的目标，**下行趋势明确**
- 从2025H2开始增速逐季放缓，Q2较Q1减速0.3pct
- 第三产业（5.2%）增速仍高于第二产业（3.9%），但差距在收窄

### 2.2 工业产出 — 有韧性但边际走弱

- **工业增加值：5.7%** — 仍处于正常区间，主要受出口相关制造业支撑
- **制造业PMI：50.3** — 仅略高于荣枯线，接近停滞
- **非制造业PMI：50.2** — 同样微弱扩张，服务业动能不足
- PMI自2026年2月触底49.0后小幅回升，但始终低于历史均值（51+），**扩张动能极弱**

### 2.3 固定资产投资 — 断崖式收缩 ⚠️

- **FAI同比：-15.6%**（最新数据，上期为-17.15%，有所收窄但仍深度负值）
- 这是**本轮周期最严重的内需崩塌信号**
- 房地产投资、基建投资、制造业投资三大分项同步收缩
- 土地出让收入同比大幅下降（数据缺失但可通过FAI推断）

### 2.4 消费 — 极度疲弱

- **零售总额同比：1.0%**（最新）— 较前期-0.6%有所回升，但仍远低于正常水平（5-8%）
- **消费者信心指数：89.9** — 低于100的荣枯线，反映居民对未来极度悲观
- 消费疲弱的根源：房地产财富效应消失 + 就业预期恶化 + 收入增长放缓

### 2.5 外贸 — 唯一亮点

- **出口同比：7.2%** — 保持较强劲增长
- **进口同比：4.1%** — 弱于出口，反映内需不足
- **贸易顺差：982.4亿美元** — 创记录高位
- 出口是当前GDP的唯一正贡献因子，但面临**全球贸易摩擦加剧**的风险

## 三、通胀维度分析

### 3.1 消费者价格 — 零通胀/轻度通缩

| 指标 | 数值 |
|:---|:---:|
| CPI同比 | **0.0%** |
| CPI环比 | +0.4%（季节性因素） |
| 核心CPI | 估计更低 |

- CPI连续多月在0%附近徘徊，**本质上是通缩**
- 2026年6月CPI同比仅1.0%（高于2025年），但剔除食品能源后的核心通胀极低
- 环比数据波动主要受季节性/天气因素影响

### 3.2 生产者价格 — 深度通缩

- **PPI同比：-3.6%** — 连续20+个月负值
- 工业部门处于典型通缩状态
- 企业利润率承压，实际融资成本（名义利率 - PPI）居高不下
- **这是抑制企业投资意愿的核心原因之一**

### 3.3 结构性通胀分化

- **农产品价格：-4.9% yoy**（CGPI Agricultural）— 食品通缩加剧
- **能源价格：+17.7% yoy**（CGPI Energy）— 输入性能源成本压力
- 上游（能源）涨、下游（消费品/农产品）跌，**企业利润进一步被挤压**

## 四、政策维度分析

### 4.1 货币政策 — 极度宽松但传导失效

| 指标 | 数值 | 信号 |
|:---|:---:|:---:|
| LPR 1Y | 3.0% | 历史低位 |
| SHIBOR 3M | 1.43% | 极低流动性充裕 |
| M2同比 | 8.8% | 货币供给充裕 |
| 10Y国债收益率 | 1.74% | 反映经济悲观预期 |
| 10Y-2Y利差 | 0.47% | 牛陡形态 |

**核心矛盾：M2高增（8.8%）但与信用创造严重脱节。**

- 新贷款同比：**-25.2%** — 信贷需求断崖式萎缩
- 社融存量大但增量枯竭
- 居民和企业都在**主动去杠杆**（提前还贷、收缩投资）
- 流动性陷入"推绳子"困境 — 央行宽松但没人借钱

### 4.2 财政政策 — 空间有限

- 地方政府土地收入锐减+债务约束，财政扩张能力受限
- 专项债发行节奏偏慢，项目储备不足
- 实际财政乘数下降

### 4.3 汇率

- USD/CNY：6.768（近期小幅升值）
- 人民币面临贬值压力但央行通过中间价维稳
- 出口强劲提供了一定的汇率支撑

## 五、综合场景诊断

### 5.1 核心叙事：资产负债表衰退

| 传导链条 | 状态 |
|:---|:---:|
| 房地产下行 → 居民财富缩水 | ✅ 发生中 |
| 居民收入预期恶化 → 消费降级 | ✅ 发生中 |
| 企业盈利下降 → 裁员/降薪 | ✅ 发生中 |
| 信贷需求萎缩 → 货币宽松失效 | ✅ 发生中 |
| 通缩预期自我强化 → 推迟消费/投资 | ✅ 形成中 |
| 出口下行风险 → 最后支撑可能崩塌 | ⚠️ 潜在风险 |

### 5.2 与历史类比

| 特征 | 当前中国 | 日本1990s |
|:---|:---:|:---:|
| 资产价格下跌 | ✅ | ✅ |
| 通缩/低通胀 | ✅ | ✅ |
| 货币宽松失效 | ✅ | ✅ |
| 信贷收缩 | ✅ | ✅ |
| 出口支撑 | ✅ | ✅（前期） |
| 人口老龄化 | ✅ | ✅ |

**相似度极高，但中国有更强的政策空间和更大的经济体量缓冲。**

### 5.3 风险情景

**上行风险（15%概率）：**
- 中央政府超预期大规模财政刺激（如直接发钱/消费券）
- 房地产政策重大转向（政府收购烂尾楼、直接入市托底）
- 出口持续超预期（但面临关税风险）

**下行风险（30%概率）：**
- 出口受全球贸易摩擦冲击（美国大选后关税升级）
- 通缩-债务螺旋深化，地方债/城投债违约事件触发系统性风险
- 房地产下行进入第二波（房企债务危机再爆发）

## 七、核心结论

**当前中国宏观场景为「通缩性停滞」（Disinflationary Stagnation），处于典型的资产负债表衰退初期至中期阶段。**

- **增长**：GDP 4.7%低于目标，内需（消费+投资）全面收缩，出口是唯一支撑
- **通胀**：CPI 0.0%/PPI -3.6%双通缩，实际利率偏高抑制经济活力
- **政策**：货币极度宽松但信用创造断裂（新贷款-25%），财政传导不畅
- **关键转折信号**：需关注中央政府是否推出直达居民部门的财政刺激（如消费券、现金转移支付），或地产行业出现系统性政策转向

**置信度：85%。** 核心数据（GDP/CPI/FAI/贷款）方向一致、相互印证，构建了高度一致的宏观图景。主要不确定性在于出口韧性能维持多久，以及政策层面可能的超预期变化。

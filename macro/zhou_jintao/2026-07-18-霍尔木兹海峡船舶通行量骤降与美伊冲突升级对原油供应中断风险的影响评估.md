# 霍尔木兹海峡船舶通行量骤降与美伊冲突升级：原油供应中断风险影响评估

## 二、战略重要性评估

### 2.1 霍尔木兹海峡的战略地位

霍尔木兹海峡是全球**最重要的能源咽喉要道**：

- 承载全球约 **20-25%** 的石油运输量（每天约1700-2100万桶）
- 是中东原油（沙特、伊拉克、阿联酋、科威特、伊朗、卡塔尔）出口的必经之路
- 全球约 **30%** 的液化天然气（LNG）贸易也经由此海峡
- 替代路线极其有限：仅剩的绕行好望角方案将增加数周航程和巨额运输成本

### 2.2 当前冲突的严重性

此次事件的严重性体现在多个"前所未有"的维度：

1. **美军连续多晚打击伊朗**：7晚的持续军事打击是近年来美伊冲突中最持久、最密集的军事行动
2. **海上封锁**：美军明确表示执行海上封锁，这本质上是战争行为
3. **实际船只遇袭**：泰国货轮在海峡遭袭，证明军事冲突已波及民用航运
4. **伊朗的升级威胁**：从"防御"转向"全面进攻与摧毁"的表述，表明伊朗可能采取更激进的应对措施
5. **通行量骤降**：船舶通过量降至三周最低点，说明航运公司已经开始主动规避风险

## 三、影响传导路径分析

### 3.1 原油市场：最直接的影响

```
霍尔木兹海峡受限 → 中东原油出口受阻 → 全球原油供应缺口
    → 油价急剧上涨 → 通胀压力上升 → 央行政策紧缩风险
    → 航运保险费用飙升 → 运输成本全面上涨
    → 能源进口国（中日韩印欧）受影响最大
```

**原油市场的关键风险**：
- 如果霍尔木兹海峡通行持续受阻，布伦特/WTI原油可能在数周内飙升30-50%甚至更高
- 战略石油储备（SPR）释放可能短期缓解，但无法替代霍尔木兹的长期通行
- OPEC剩余产能（主要在沙特和阿联酋）也受制于海峡通行，无法有效填补缺口

### 3.2 全球通胀：第二波冲击

美国6月CPI为 **3.5%**（数据截至2026-06），核心CPI为0.3%，通胀尚未回到2%目标。**原油价格飙升将直接推高能源成本**，并传导至运输、化工、制造业等各个环节：

- 汽油价格上涨将直接影响消费者信心和实际购买力
- 美国6月消费者信心指数为97.4（较5月89.9回升），但若油价持续上涨可能再度恶化
- 欧洲和亚洲能源进口国面临更严重的输入性通胀

### 3.3 金融市场

当前观察到：
- **美股三大指数集体下跌**：市场已经在定价地缘政治风险
- **费城半导体指数进入熊市**（较高位回撤超20%）：科技股对经济增长预期极度敏感
- 美国10年期国债收益率 **4.57%**，2年期 **4.16%**，利差0.41%仍为正：但如果通胀预期恶化，收益率曲线可能再度倒挂

### 3.4 美联储政策路径

- 当前通胀（CPI 3.5%）尚未达标，油价冲击可能导致通胀反弹
- 美联储面临**滞胀困境**：油价冲击推高通胀的同时抑制经济增长
- 若油价持续高位，美联储可能被迫推迟降息甚至重新考虑加息

## 四、情景分析

### 情景一：短期冲突（概率35%）
- 冲突在未来1-2周内缓和，海峡恢复正常通行
- 油价短期冲高后回落，涨幅10-15%
- 市场消化风险后企稳
- **影响：可控**

### 情景二：持续封锁（概率45%）
- 美伊军事对峙持续3-6周，海峡通行持续受限
- 油价上涨30-50%，全球通胀回升
- 美联储无法降息，全球经济增长放缓
- 亚洲能源进口国受到严重冲击
- **影响：重大**

### 情景三：全面升级（概率20%）
- 冲突扩大为地区性战争，伊朗封锁海峡或大量布雷
- 霍尔木兹海峡实质关闭，全球原油供应出现每日百万桶级别缺口
- 油价翻倍，全球性经济危机
- 各国SPR迅速消耗，可能引发能源配给
- **影响：灾难性**

## 六、结论与建议

### 核心判断

**霍尔木兹海峡船舶通行量骤降是一个真实且正在发酵的宏观叙事**，其背后是美伊军事冲突的急剧升级。当前强度0.50的判断合理，但风险偏向于进一步上升。

**这是当前全球市场最具影响力的地缘政治风险因子**，其影响力超过贸易摩擦、科技管制等其他宏观叙事。

### 建议的行动

1. **立即纳入宏观监测框架**：将霍尔木兹海峡船舶通行量、WTI/布伦特原油价格、美伊外交动态列为每日必跟踪指标
2. **评估投资组合风险敞口**：高度关注能源进口国（日本、韩国、印度、中国、欧洲）的资产风险
3. **关注受益标的**：能源股、航运股（非海峡航线）、国防军工、新能源可能受益
4. **警惕尾部风险**：情景三（全面升级）的尾部风险虽概率低但影响极大，需有对冲预案

### 置信度

- 置信度：**0.65**（中等偏高）
  - 事件真实性已通过多源交叉验证 ✓
  - 但当前海峡通行量下降是主动规避还是被动封锁尚需确认
  - 冲突后续演变路径的不确定性较大

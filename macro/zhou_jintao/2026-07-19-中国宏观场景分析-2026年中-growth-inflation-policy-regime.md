## 2. Growth（经济增长）：**显著低于潜力，趋势恶化**

### 2.1 GDP增速下台阶

- **H1 2026 GDP同比 4.7%**，较2025年全年5.0%下行0.3个百分点
- **Q1 2026 GDP同比 5.0%** → **H1 2026 4.7%**，仅Q2单季减速更明显（估算约4.4%）
- 趋势线：2023年疫情修复（5.2%）→ 2024年（5.0%）→ 2025年（5.0%）→ 2026H1（4.7%）

**结论**：GDP增速已跌破5%的关键心理/政策底线，且下行动能未见底。按季度来看，2026年第二季度减速幅度扩大。

### 2.2 PMI：荣枯线附近挣扎

| 月份 | 制造业PMI | 非制造业PMI |
|------|----------|------------|
| 2026年2月 | 49.0 ❌ | 49.5 ❌ |
| 2026年3月 | 50.4 | 50.1 |
| 2026年4月 | 50.3 | 49.4 ❌ |
| 2026年5月 | 50.0 ⚠️ | 50.1 |
| 2026年6月 | 50.3 | 50.2 |

过去5个月中，PMI有3个月触及或低于荣枯线。50.3的制造业PMI仅表示勉强扩张，远低于52以上的健康水平。

### 2.3 需求侧全面疲弱

| 需求侧指标 | 数值 | 解读 |
|-----------|------|------|
| **零售总额同比** | +1.0% | 消费几乎零增长，远低于疫情前8%+ |
| **固定资产投资同比** | -15.6% | **深度收缩**，房地产+基建+制造业投资全面下滑 |
| **新贷款同比** | -25.2% | 实体经济融资需求崩塌 |
| **消费者信心** | 89.9 | **低于100=悲观**，消费者不愿消费 |

**关键信号**：FAI同比-15.6%和新贷款同比-25.2%是两个极度危险的数值。这不仅仅是"放缓"，而是**信贷-投资循环正在断裂**。企业不愿借、银行放不出、投资起不来的负反馈已形成。

### 2.4 增长判断

> **场景判定：Growth Recession（增长衰退）**
> - 增速低于潜在水平（估计5.0%+）
> - 趋势明确向下（从5.3%→5.0%→4.7%）
> - 没有见底信号，信贷和投资仍在加速恶化
> - 置信度：**高（85%）**

## 3. Inflation（通胀）：**结构性分化——CPI温和 vs PPI成本推动**

### 3.1 CPI：低位运行

| 月份 | CPI同比 |
|------|--------|
| 2026年2月 | +1.3% |
| 2026年3月 | +1.0% |
| 2026年4月 | +1.2% |
| 2026年5月 | +1.2% |
| 2026年6月 | **+1.0%** |

CPI连续6个月在1.0%-1.3%区间，**远低于3%的政策目标**。环比数据（2026年6月-0.3%）显示物价在边际走弱。这不是通缩（deflation），但属于**明显的低通胀（lowflation）**。

### 3.2 PPI：从通缩到成本推动型通胀

| 月份 | PPI同比 | 趋势 |
|------|--------|------|
| 2026年2月 | -0.9% | 仍通缩 |
| 2026年3月 | +0.5% | 转正 |
| 2026年4月 | +2.8% | 加速 |
| 2026年5月 | +3.9% | 加速 |
| 2026年6月 | **+4.1%** | 持续走高 |

PPI从2026年初的通缩（-0.9%）快速攀升至+4.1%，驱动因素是**能源价格暴涨**（能源CGPI同比+17.7%），而农业CGPI仍在通缩（-4.9%）。

### 3.3 CPI-PPI剪刀差——企业利润挤压

```
CPI同比: +1.0%
PPI同比: +4.1%
CPI-PPI剪刀差: -3.1个百分点 ← 对中下游企业极不利
```

这意味着：
- **上游**：能源等原材料价格大幅上涨（+17.7%）
- **中下游**：由于终端需求疲弱（零售+1.0%），企业无法将成本转嫁
- **结果**：中下游制造业利润率被压缩，这解释了为什么企业不愿扩大生产和投资

### 3.4 通胀判断

> **场景判定：结构性通胀分化（Divergent Inflation）**
> - CPI：低通胀（~1.0%），反映需求不足
> - PPI：成本推动型通胀（4.1%），能源驱动
> - 这不是统一的价格上涨，而是"需求弱+成本高"的双重挤压
> - 置信度：**高（80%）**

## 4. Policy（政策）：**极度宽松但传导失灵**

### 4.1 货币政策：历史性宽松

| 指标 | 当前值 | 历史对比 |
|------|-------|---------|
| LPR 1Y | 3.0% | 历史最低附近 |
| SHIBOR 3M | 1.43% | 极低水平 |
| 10Y国债收益率 | 1.74% | 历史低位（反映衰退预期） |
| 10Y-2Y利差 | 0.48% | 正但平坦 |

三个利率同时处于历史低位或接近历史低位，说明：
1. 央行已持续降息降准
2. 银行间流动性极其充裕
3. 债券市场定价了显著的衰退预期（10Y国债仅1.74%）

### 4.2 信用传导断裂——"推绳子"困境

> **流动性充裕 ≠ 信用扩张**

关键矛盾数据：

| 宽松面 | 实体经济面 |
|--------|-----------|
| SHIBOR 3M = 1.43%（极低） | 新贷款同比 = -25.2%（崩塌） |
| 10Y国债 = 1.74%（极低） | FAI同比 = -15.6%（深度收缩） |
| LPR = 3.0%（历史低位） | 消费者信心 = 89.9（悲观） |

**机制分析**：
- 银行间利率再低，企业也不愿借钱（信心不足，投资回报率预期低）
- 居民不愿消费（信心89.9），不借钱也不花钱
- 房地产行业仍在去杠杆，不贡献信用需求
- 地方债务约束下，基建投资也难发力

这是典型的**流动性陷阱/资产负债表衰退**特征——货币政策提供了弹药，但没有扣扳机的人。

### 4.3 财政政策

从数据看，固定资产投资-15.6%说明财政扩张力度不足，或地方政府的实际执行能力受限。社融总量6,245亿元（2026年4月）看似不低，但结构上可能主要是政府债券和票据融资，而非实体经济中长期贷款。

### 4.4 政策判断

> **场景判定：Accommodative but Impotent（宽松但无力）**
> - 货币政策：✅ 已极度宽松
> - 传导机制：❌ 断裂，信用需求崩塌
> - 政策有效性：低——继续降息可能边际效果递减
> - 置信度：**高（85%）**

## 5. 综合宏观场景分类

### 5.1 三维度评分

| 维度 | 评分 | 标签 |
|------|------|------|
| **Growth** | ⬇️ 4/10 | 增长衰退，低于潜力，趋势恶化 |
| **Inflation（CPI）** | ➡️ 2/10 | 低通胀，需求不足主导 |
| **Inflation（PPI）** | ⬆️ 7/10 | 成本推动通胀，能源驱动 |
| **Policy** | ⬇️ 3/10 | 极度宽松但传导失灵 |

### 5.2 场景分类结论

> ## 主场景：Growth Recession + Lowflation（增长衰退 + 低通胀场景）
>
> **置信度：80%**

**核心理由**：
1. GDP增速下破5%且持续减速（4.7%→更低）
2. 消费（+1.0%）、投资（-15.6%）、信贷（-25.2%）全面走弱
3. CPI仅+1.0%，反映内需严重不足
4. 政策虽宽松但传导受阻

> ## 子场景/风险：Cost-Push Squeeze（成本推动型挤压）
>
> **置信度：60%**

**核心理由**：
1. PPI升至+4.1%且加速，能源价格+17.7%
2. CPI-PPI剪刀差-3.1pct，中下游利润承压
3. 如果能源价格继续上行，可能演化成"类滞胀"压力（增速下行+成本推动通胀）

### 5.3 排除的替代场景

| 被排除的场景 | 排除理由 |
|-------------|---------|
| **V型复苏** | 信贷和投资仍在恶化，无复苏迹象 |
| **硬着陆/危机** | PMI仍在50以上，银行体系稳定，外汇储备充足 |
| **全面通缩** | CPI仍为正，PPI已转正 |
| **过热/繁荣** | 增长和通胀均不支持 |

## 6. 资产含义与关键风险

### 6.1 资产含义

| 资产类别 | 在当前场景下的含义 |
|---------|-----------------|
| **中国利率债** | 增长衰退+低通胀=债牛延续。10Y国债1.74%仍有下行空间 |
| **中国信用债** | 信用分化加剧，中下游企业（PPI成本挤压）需谨慎 |
| **A股** | 盈利预期下修（PPI挤压+销售放缓），防御性配置占优 |
| **人民币** | 增长偏弱+利差收窄=贬值压力，但监管可控 |
| **商品（能源）** | 能源CGPI+17.7%，短期偏强但需求端不支持持续 |
| **房地产** | FAI-15.6%、新房贷款-25%——行业调整远未结束 |

### 6.2 关键监测指标

1. **每月PMI**：是否跌破50确认衰退深化
2. **新贷款同比**：信贷收缩是否触底
3. **PPI-CPI剪刀差**：是否继续扩大（中下游利润）
4. **消费者信心**：是否回升至95以上（消费复苏前提）
5. **政策信号**：是否出现非常规宽松（特别国债、财政直接补贴居民）

### 6.3 尾部风险

- **能源价格持续飙升**推动PPI>5%，叠加GDP<4.5%，演化为类滞胀
- **房地产信用事件扩散**至金融体系
- **地方政府财政危机**导致基建投资进一步收缩

## 7. 总结

当前中国经济处于 **"增长衰退+低通胀+宽松传导失灵"** 的非典型宏观场景。经济增长（~4.7%）显著低于潜在水平且趋势恶化，消费、投资、信贷三个发动机同时熄火。通胀层面呈现结构性分化：CPI因需求不足维持低位（~1.0%），但能源驱动PPI升至4.1%挤压中下游。货币政策已极度宽松（LPR 3.0%, SHIBOR 1.43%），但信用传导机制断裂——这是央行"推绳子"的经典困境。

**核心矛盾**：不是流动性不够，而是信心不足和需求不足。下一次政策应对可能需要从货币宽松转向**直接的财政扩张和居民端补贴**，才能打破"不敢消费→企业不投资→就业恶化→更不敢消费"的负循环。

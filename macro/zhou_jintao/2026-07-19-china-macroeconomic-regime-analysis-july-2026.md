## 二、数据支撑

### 2.1 增长维度：明显放缓，结构严重分化

#### GDP增速持续下行

从5.3%→5.0%→4.7%，GDP增速下台阶趋势确认。2026年H1的4.7%是近年来较低水平（剔除2020/2022疫情年份）。

### 2.2 通胀维度：结构性分化，整体疲弱

#### CPI：持续低位运行

| 月份 | CPI同比 |

| 2026年3月 | 1.0% |
| 2026年4月 | 1.2% |

CPI连续5个月在1.0%-1.3%区间，远低于央行3%的警戒线。6月环比-0.3%更显示通缩压力在加剧。核心CPI如果剔除能源相关传导，可能更低。

#### PPI：从通缩回升，但成本驱动非需求驱动

| 月份 | PPI同比 |
|------|---------|
| 2026年2月 | -0.9% |
| 2026年3月 | 0.5% |
| 2026年4月 | 2.8% |
| 2026年5月 | 3.9% |
| **2026年6月** | **4.1%** |

PPI从年初通缩快速回升至4.1%，但这是能源和商品价格驱动，而非内需拉动。CGPI能源同比**17.7%** vs CGPI农业同比 **-4.9%** 形成鲜明对比，说明通胀压力完全来自供给侧（能源），而需求侧（农业、消费品）仍在通缩。

**通胀判断：整体低通胀偏通缩，PPI回升不具可持续性。**

## 三、综合判断

### 核心矛盾：两速分化 + 传导失灵

中国经济呈现典型的 **"两速分化"** 格局：

```
外需部门（出口）  ⚡ 强劲（+27%）
内需部门（消费）  🐢 停滞（+1%）
投资部门（FAI）   💀 收缩（-15.6%）
```

货币政策的传导机制已严重受损——低利率无法转化为信贷增长和投资扩张。这种状况在经济学上接近 **"资产负债表衰退"** 的特征：微观主体（居民、企业）在去杠杆而非加杠杆，无论利率多低都不愿借钱。

### 与历史类比

当前中国的宏观状态与以下历史时期有相似性：
- **日本1990年代**：资产泡沫破裂后，低利率+信用收缩+通缩压力
- **中国2015年**：股灾后经济减速，但当时有房地产和基建作为对冲工具
- **区别**：当前房地产仍在筑底，地方政府债务压力制约财政扩张空间

### 未来3-6个月展望

1. **增长**：H2 2026 GDP可能在4.5%-4.8%区间，全年大概率在4.7%-4.9%，低于政府目标（通常5%左右）
2. **通胀**：CPI将继续在1%附近运行，通缩风险不可忽视
3. **政策**：预计会有进一步的降息降准，但核心问题是信用传导而非利率水平。财政政策发力（专项债、特别国债）可能比货币政策更有效
4. **风险因素**：出口如果因全球需求放缓而走弱，经济将失去最后一根支柱

### 投资启示

- **债券**：低增长+低通胀利于债券，但1.74%的10Y利率已计入较多悲观预期
- **权益**：内需疲弱背景下，与国内消费/投资相关的板块承压，出口/制造业出海主题相对占优
- **汇率**：人民币升值空间有限，但大幅贬值概率也不高（出口强+政策维稳）
- **商品**：内需驱动的商品（黑色、建材）承压，能源类受外溢影响
